# 印花税

## 概述

印花税是对经济活动中设立、领受具有法律效力凭证的行为所征收的一种税。因其在凭证上粘贴印花税票作为完税标记而得名，属于行为税范畴。目前中国是全球少数仍征收证券交易印花税的主要经济体之一。

## 定义与基本特征

- **定义**：对在经济交往中设立、领受具有法律效力凭证的行为征收的税种
- **性质**：行为税（对特定行为课税，非对所得或财产本身）
- **特点**：
  - 比例税率与定额税率并行
  - 从价计征与从量计征并行
  - 纳税人自行计算、自行购买、自行贴花（"三自"原则）
  - 凭证具有法律效力是课税前提

## 历史起源

### 国际起源

- **1624年，荷兰**首创印花税，政府在财政困难时寻求新税源
- **1694年，英国**开始征收，随后传播到欧洲各国
- **理论依据**：税负不易转嫁——对文书（凭证）课税，最终由签署人承担

### 中国历史

| 时期 | 事件 |
|------|------|
| 清末 | 由欧洲传入，1902年开始酝酿 |
| 1912年 | 中华民国颁布首部《印花税法》 |
| 1950年 | 政务院发布《印花税暂行条例》 |
| 1958年 | 并入工商统一税一度停征 |
| 1988年 | 国务院重发《印花税暂行条例》恢复征收 |
| 2021年 | 《印花税法》通过，2022年7月1日起施行 |

## 中国印花税法：征税范围

### 《印花税法》（2022年施行）规定的应税凭证

**合同类（11类）**
买卖合同、借款合同、融资租赁合同、租赁合同、承揽合同、建设工程合同、运输合同、技术合同、保管合同、仓储合同、财产保险合同

**产权转移书据**
- 土地使用权出让书据
- 土地使用权、房屋等建筑物和构筑物所有权转让书据
- 股权转让书据（不包括股票投资）
- 商标专用权、著作权、专利权、专有技术使用权转让书据

**营业账簿**
记载实收资本（股本）、资本公积的账簿

**证券交易**
在证券交易所和国务院批准的其他证券交易场所转让股份、债券、存托凭证等

## 豁免情况

- 应税凭证的副本或抄本（正本已贴花的除外）
- 农民、家庭农场等购买农业生产资料或销售农产品书立的买卖合同和保险合同
- 无息/贴息借款合同、国际组织优惠性贷款安排
- 财产赠与政府、学校、社会福利机构、慈善组织书立的产权转移书据
- 非营利性医疗卫生机构采购药品或卫生材料书立的买卖合同
- 电子商务经营者与用户订立的电子订单

## 经济影响分析

### 正面作用
- **财政功能**：为政府提供稳定收入（2021年证券交易印花税达2,048亿元）
- **抑制投机**：提高交易成本，理论上抑制过度投机
- **信号功能**：印花税调整往往被视为政策态度的风向标

### 负面影响
- **降低流动性**：增加交易成本，抑制正常交易活动
- **累退性**：对小额频繁交易的个人投资者影响更大
- **国际竞争力**：相比无印花税市场，可能降低本地市场吸引力
- **市场扭曲**：可能改变投资者交易决策，造成资源配置扭曲

## 主要争议

1. **是否取消证券交易印花税**
   - 支持者（如余永定等学者）：取消以活跃资本市场
   - 反对者：财政收入减少，市场可能过度投机

2. **调控效果存疑**
   - 实证研究表明，印花税调整对市场的影响往往是短暂的
   - 长期走势取决于基本面而非交易成本

3. **改革方向讨论**
   - 逐步降低税率而非一次性取消
   - 改为类似美国SEC收费模式
   - 适时推出资本利得税替代
