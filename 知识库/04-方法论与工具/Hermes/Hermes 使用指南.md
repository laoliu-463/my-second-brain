# Hermes 使用指南

## 概述

Hermes 本质是一个"找资料"和"整理信息"的 AI 助手，而非存储大量电子书的资料库。核心用法是通过自然语言下指令，引导它查询特定在线资源，并利用内置技能系统（Skills）完成信息处理任务。

## 第一步：掌握核心指令

直接在对话界面用自然语言下指令，例如：

```
"请帮我搜索并总结微观经济学中关于供需理论的核心观点。"
```

## 第二步：为不同学科指定优质资源

| 学科领域 | 推荐首选资源 | 优质备选资源 | 搜索关键词建议 |
| :--- | :--- | :--- | :--- |
| **微观经济学** | 国家高等教育智慧教育平台 | 曼昆《经济学原理》（微观分册）、范里安《微观经济学：现代观点》 | 微观经济学、在线课程、教材、课件 |
| **宏观经济学** | 国家高等教育智慧教育平台 | 曼昆《宏观经济学》、国家统计局数据库 | 宏观经济学、在线课程、曼昆、PDF |
| **心理学 (马斯洛)** | 超星尔雅等 MOOC 平台 | 马斯洛《动机与人格》 | 马斯洛、需求层次理论、动机与人格 |
| **历史学 (竹书纪年)** | 《古本竹书纪年辑证》等现代辑本 | 明代范钦订本影印本 | 竹书纪年、古本竹书纪年辑证、古籍、PDF |
| **政治经济学** | 国家高等教育智慧教育平台 | 逄锦聚版《政治经济学》、《资本论》导读 | 政治经济学、在线课程、资本论、教材 |

## 第三步：使用技能组合（Skills Combo）

Hermes 最强大的地方是技能系统。解决"找资料+写总结"类长任务，用组合拳效果最好：

1. **先用 `web_search` 找资料**
   ```
   "使用 web_search 技能，在'国家高等教育智慧教育平台'搜索'微观经济学'的完整课程大纲和视频资源链接。"
   ```

2. **再用 `documents` 或 `wiki` 技能做整理**
   ```
   "请利用文档技能，将刚才找到的关于'马斯洛需求层次理论'的材料，整理成一份包含七层次定义的 Markdown 格式笔记，并保存。"
   ```

3. **创建专属技能复用流程**
   ```
   "请为我创建一个专属技能，名字叫'宏观数据简报'，它能自动搜索国家统计局最新的 CPI、GDP 和 PMI 数据，并生成一个表格。"
   ```

## 第四步：通用资料搜索示例

- **找经济学课程**
  ```
  "请使用 web_search 技能，在'国家高等教育智慧教育平台'等网站上，查找免费的本科《微观经济学》和《宏观经济学》中文课程，并列出它们的课程大纲和链接。"
  ```

- **找心理学理论介绍**
  ```
  "请帮我查网络资料并介绍马斯洛需求层次理论的七个层次，并为每个层次举一个生活中的例子。"
  ```

- **找历史文献资料**
  ```
  "请在网上查找《古本竹书纪年辑证》，并列出其关于夏朝和商朝部分的目录。"
  ```

- **找政治经济学经典**
  ```
  "请搜索并推荐 3-5 本《政治经济学》领域的经典教材，并注明作者，如果可以，请提供它们的在线阅读或下载链接。"
  ```

## 关键原则

- **核心能力**：持续学习和流程自动化
- **关键**：给 Hermes 设定明确的任务边界和资源来源
- **最优资源**：免费在线课程（MOOC）和权威数据库是首选
