## 一、微观经济学

### 1.1 什么是微观经济学

微观经济学以单个经济主体（消费者、企业）为研究对象，研究资源配置效率问题。核心问题是：在资源稀缺的前提下，如何实现效用最大化和利润最大化？

**与宏观经济学的区别：**
- 微观：看树木（单个市场、单个消费者、单个企业）
- 宏观：看森林（GDP、通货膨胀、失业、整个经济运行）

---

### 1.2 供求理论

#### 需求（Demand）

**需求定律：** 价格上升，需求量下降；价格下降，需求量上升（排除特殊情况）。

**需求的影响因素：**
- 商品本身价格
- 消费者收入
- 替代品价格
- 互补品价格
- 消费者偏好
- 预期（对未来价格的预期）
- 人口因素

**需求函数：** Qd = f(P)，表示需求量与价格之间的函数关系。

**需求的价格弹性（Ed）：**
```
Ed = 需求量变动百分比 / 价格变动百分比

|Ed| > 1：富有弹性（奢侈品）
|Ed| = 1：单位弹性
|Ed| < 1：缺乏弹性（必需品）
Ed = 0：完全无弹性（救命药）
Ed = ∞：完全弹性（完全竞争市场的单个厂商）
```

**生活中的例子：**
- 矿泉水：需求弹性小（便宜又必需）
- iPhone：需求弹性大（可替代品多）

#### 供给（Supply）

**供给定律：** 价格上升，供给量上升；价格下降，供给量下降。

**供给的影响因素：**
- 商品本身价格
- 生产成本（原材料、工资、技术）
- 生产技术水平
- 预期
- 生产者数量

**供给的价格弹性（Es）：**
```
Es = 供给量变动百分比 / 价格变动百分比

Es > 1：富有弹性
Es = 1：单位弹性
Es < 1：缺乏弹性
```

#### 均衡价格

**均衡：** 供给曲线与需求曲线相交的点，此时需求量 = 供给量。

**均衡价格的形成：**
- 价格高于均衡点 → 供给 > 需求 → 价格下降
- 价格低于均衡点 → 需求 > 供给 → 价格上升
- 市场自动调节，最终趋向均衡

**政府干预——最高限价与最低限价：**
- 最高限价（Price Ceiling）：低于均衡价格 → 导致短缺（排队、配给）
  - 例：房租管制 → 租房难
- 最低限价（Price Floor）：高于均衡价格 → 导致过剩
  - 例：最低工资标准 → 失业增加

### 1.3 消费者行为理论

#### 效用论

**基数效用（已过时，仅作了解）：**
- 效用（Utility）可以量化
- 边际效用递减：每增加一单位消费，边际效用递减
- 消费者均衡条件：MU1/P1 = MU2/P2 = ... = λ（最后一元钱的边际效用）

**序数效用（主流）：**
- 效用不能量化，只能比较偏好顺序
- 用无差异曲线表示

#### 无差异曲线

**定义：** 给消费者带来相同满足程度的不同商品组合的轨迹。

**无差异曲线的特征：**
1. 向右下方倾斜（边际替代率递减）
2. 凸向原点（凸性）
3. 不相交
4. 离原点越远，偏好越高

**边际替代率（MRS）：**
```
MRSxy = 减少X的数量 / 增加Y的数量
= - dX/dY（沿无差异曲线）
```

#### 消费者均衡

**预算约束线：** I = Px·X + Py·Y

**消费者均衡点：** 无差异曲线与预算线相切之点。

**条件：** MRSxy = Px/Py
（即：消费者花费每一元钱购买任意两种商品所带来的边际效用相等）

#### 收入效应与替代效应

**某种商品价格下降时：**
- **替代效应（Substitution Effect）：** 该商品相对变便宜了，消费者会用它替代其他商品 → 需求量增加
- **收入效应（Income Effect）：** 实际购买力上升，消费者能买更多 → 需求量增加

**正常品 vs 低档品：**
- 正常品：收入效应使需求同方向变动
- 低档品：收入效应使需求反方向变动（但替代效应始终使需求同方向变动）

### 1.4 生产者行为理论

#### 生产函数

**定义：** Q = f(L, K)，即产出量与投入要素之间的技术关系。

**短期 vs 长期：**
- 短期：至少有一种投入要素固定（通常假设资本K固定）
- 长期：所有投入要素均可调整

#### 短期生产函数

**总产量（TP）：** Q = f(L)

**边际产量（MP）：**
```
MP = dTP/dL = ΔQ/ΔL
```

**平均产量（AP）：**
```
AP = TP/L
```

**边际报酬递减规律（关键！）：**
- 在技术水平不变的条件下，连续追加可变要素L的投入：
  - 开始时 MP 递增
  - 继续追加，MP 开始递减
  - 最终 MP 可能为负

**三个阶段：**
```
阶段Ⅰ：MP > AP → AP递增（厂商不会停在此阶段）
阶段Ⅱ：MP < AP，但MP > 0 → AP递减（理性生产区间）
阶段Ⅲ：MP < 0 → 总产量绝对减少（不可能在此阶段生产）
```

#### 长期生产函数——等产量曲线

**等产量曲线（Isoquant）：** 生产相同产量的不同要素组合轨迹。

**边际技术替代率（MRTS）：**
```
MRTSLK = - dK/dL = MP_L / MP_K
```

**规模报酬（Returns to Scale）：**
- 规模报酬递增：所有要素同比例增加，产量增加更多（1+1>2）
- 规模报酬不变：等比例增加，产量同比例增加（1+1=2）
- 规模报酬递减：等比例增加，产量增加更少（1+1<2）

### 1.5 成本理论

#### 短期成本

| 成本类型 | 简写 | 定义 |
|----------|------|------|
| 总固定成本 | TFC | 不随产量变化的成本（厂房、设备） |
| 总可变成本 | TVC | 随产量变化的成本（原材料、劳动力） |
| 总成本 | TC | TC = TFC + TVC |
| 平均固定成本 | AFC | AFC = TFC/Q（随产量增加而递减） |
| 平均可变成本 | AVC | AVC = TVC/Q |
| 平均总成本 | AC | AC = TC/Q = AFC + AVC |
| 边际成本 | MC | MC = ΔTC/ΔQ = dTC/dQ（核心！） |

**MC曲线的特征：**
- 先下降，后上升（U形）
- MC交于AC和AVC的最低点
- MC < AC时，AC下降；MC > AC时，AC上升

#### 长期成本

**长期成本曲线：** 由无数条短期成本曲线（短期平均成本SAC）的包络线形成。

**LAC（长期平均成本）的特征：**
- 通常呈U形（规模经济→规模不经济）
- 不存在固定成本概念

### 1.6 市场结构理论

#### 完全竞争市场

**特征：**
1. 大量买者和卖者
2. 产品同质（无差别）
3. 资源完全流动
4. 信息完全对称
5. 无进入退出壁垒

**完全竞争厂商的需求曲线：**
- 单个厂商是价格接受者（P = MR = AR）
- 需求曲线 = 边际收益线 = 平均收益线，均为水平线

**短期均衡：** P > AC → 盈利；P = AC → 正常利润；P < AC → 亏损（但P > AVC时继续生产）

**长期均衡条件：** MR = AR = MC = AC（P = AC的最低点，盈利为零）

#### 垄断市场

**特征：**
1. 只有一个厂商
2. 产品没有紧密替代品
3. 进入壁垒高（资源控制、专利、政府许可、自然垄断）

**垄断厂商的需求曲线：** 向下倾斜（P > MR）

**均衡条件：** MR = MC

**垄断厂商的定价：** 根据需求曲线定价（"价格歧视"的可能性）

**类型：**
- 一级价格歧视：按每单位最高愿意支付的价格定价（理想情况）
- 二级价格歧视：按消费量定价（阶梯定价）
- 三级价格歧视：针对不同市场定不同价

#### 垄断竞争市场

**特征：**
- 很多厂商
- 产品有差别（品牌、质量、服务不同）
- 短期有垄断力，长期竞争激烈

**垄断竞争厂商的需求曲线：** 向右下方倾斜，但比垄断厂商平坦（替代性较强）

**短期均衡：** MR = MC
**长期均衡：** MR = MC，且 P = AC（盈利为零）

#### 寡头市场

**特征：**
- 少数几个厂商
- 相互依存（一家决策影响其他家）
- 进出壁垒高

**博弈论在寡头中的应用：**

**古诺模型（双寡头）：**
- 假设：两个厂商生产同质产品，产量决策
- 均衡：各占市场1/3（而非1/2），反应函数交点

**伯特兰模型（价格竞争）：**
- 均衡价格 = 边际成本（激烈价格战）

**斯塔克伯格模型：**
- 一个厂商是领导者（有先动优势），另一个是追随者

**卡特尔（串谋）：**
- 多个寡头联合起来像一个垄断者一样行动
- 容易因背叛而瓦解

---

### 1.7 要素市场理论

**引致需求：** 厂商对生产要素的需求不是直接需求，而是由消费者对最终产品的需求"引致"的。

**要素使用原则：** VMP = MC
- VMP（边际产品价值）= MP × P（边际产量 × 产品价格）
- 劳动的 VMP = MP_L × P

**要素价格决定：**
- 完全竞争要素市场：要素价格 = VMP（等于边际成本）
- 买方垄断：厂商面临向上倾斜的要素供给曲线

---

### 1.8 市场失灵

#### 外部性

**外部性（Externality）：** 一个人的行为直接影响了另一个人的福利，但没有通过市场价格反映。

**负外部性（Negative）：**
- 例：污染、噪音
- 社会成本 > 私人成本
- 解决方法：政府征税（庇古税）或明晰产权（科斯定理）

**正外部性（Positive）：**
- 例：教育、疫苗
- 社会收益 > 私人收益
- 解决方法：政府补贴

#### 公共物品

**特征：**
- 非竞争性：一人消费不减少他人消费
- 非排他性：不能排除他人消费

**例：**
- 纯公共物品：国防、路灯、基础研究
- 准公共物品：公共图书馆、公共草坪（有部分竞争性或排他性）

**搭便车问题：** 公共物品由政府提供（税收融资），否则市场供给不足。

#### 信息不对称

**逆向选择（Adverse Selection）：** 交易前信息不对称 → 劣质品驱逐优质品（柠檬市场）

**道德风险（Moral Hazard）：** 交易后信息不对称 → 一方行为改变损害另一方利益（保险、委托代理问题）

**解决方法：**
- 信号发送（Signaling）：主动传递信息（学历、证书）
- 信息甄别（Screening）：被动设计机制区分不同类型（保险中的体检）

## 二、宏观经济学

### 2.1 宏观经济学的研究对象

宏观经济学研究整个经济体系的运行，研究对象包括：
- 国内生产总值（GDP）与国民生产总值（GNP）
- 通货膨胀与失业
- 经济增长
- 财政政策与货币政策
- 国际贸易与国际收支

---

### 2.2 国民收入核算

#### GDP（国内生产总值）

**定义：** 一个国家（或地区）一定时期内，所有最终产品和服务的市场价值总和。

**理解GDP的关键：**
1. 地域原则：在该国领土范围内生产的
2. 流量：一定时期内（通常是一年）
3. 最终产品：不算中间产品（避免重复计算）
4. 市场价值：用价格计量

**GDP的核算方法：**
- **生产法：** GDP = Σ（产出 - 中间投入）
- **收入法：** GDP = 工资 + 利息 + 租金 + 利润 + 折旧 + 间接税
- **支出法（最常用）：** GDP = C + I + G + (X - M)
  - C：消费（家庭）
  - I：投资（企业）
  - G：政府购买
  - X-M：净出口

**GDP的局限：**
- 不算非市场活动（家务劳动、志愿服务）
- 不反映生活质量（环境、闲暇）
- 不算非法经济活动
- 不反映收入分配

#### 其他重要指标

**GNP（国民生产总值）：** 一国国民在国内外所创造的价值总和。
**NDP（国内生产净值）：** NDP = GDP - 折旧
**NI（国民收入）：** 工资 + 利息 + 租金 + 利润
**PI（个人收入）：** 国民收入 - 公司未分配利润 - 社会保险 + 政府转移支付
**DPI（个人可支配收入）：** PI - 个人所得税

---

### 2.3 国民收入决定理论

#### 简单国民收入决定模型

**假设条件：**
- 潜在GDP（充分就业GDP）已知
- 价格水平固定（短期）
- 利率、汇率不变
- 总供给无限（产能充足）

**总需求 = 消费 + 投资：**
```
AD = C + I
C = C₀ + c·Yd  （C₀为自发消费，c为边际消费倾向，Yd为可支配收入）
```

**均衡条件：** Y = AD（即总产出 = 总需求）

**乘数效应（Multiplier Effect）：**
```
乘数 k = 1/(1-c)  或  k = 1/MPS
（MPS = 边际储蓄倾向）
```

**举例：** c = 0.8，则 k = 5
→ 政府增加100亿支出 → GDP增加500亿

**为什么有乘数效应？**
- 初始支出 → 某人收入增加
- 增加的收入中80%用于消费
- 这80%又成为他人的收入
- 循环往复，最终效应放大约5倍

#### IS-LM模型

**IS曲线（Investment-Saving）：** 产品市场均衡
- 表示：利率(i)与产出(Y)负相关
- IS左下方：产品市场供过于求

**LM曲线（Liquidity preference-Money）：** 货币市场均衡
- 表示：利率(i)与产出(Y)正相关
- LM右下方：货币市场供过于求

**IS-LM均衡：** 两条曲线交点决定均衡利率和均衡产出。

**财政政策效果：**
- 扩张性财政政策（增加G或减税）→ IS右移 → 产出增加，但利率上升（"挤出效应"）
- 挤出效应大小取决于LM曲线斜率

**货币政策效果：**
- 扩张性货币政策（增加M）→ LM右移 → 产出增加，利率下降

#### AD-AS模型（总需求-总供给模型）

**AD曲线（总需求）：** 价格与总需求的负相关关系
-向右下方倾斜（财富效应、利率效应、汇率效应）

**SRAS曲线（短期总供给）：** 向上倾斜（黏性工资理论、黏性价格理论、错觉理论）

**LRAS曲线（长期总供给）：** 垂直线（长期产出取决于供给能力，与价格无关）

**长期均衡：** AD与LRAS交点 → 潜在GDP

### 2.4 失业与通货膨胀

#### 失业

**失业率：** 失业人数 / 劳动力总数 × 100%

**自然失业率：** 即使在长期，劳动力市场也不能达到100%就业（摩擦性失业 + 结构性失业）

**失业类型：**
1. **摩擦性失业（Frictional）：** 转换工作期间的暂时性失业（正常的）
2. **结构性失业（Structural）：** 技能不匹配或地区不匹配（需要培训或迁移）
3. **周期性失业（Cyclical）：** 经济衰退导致（凯恩斯失业）

**奥肯定律（Okun's Law）：**
```
失业率每高于自然失业率1%，实际GDP低于潜在GDP约2%
```

#### 通货膨胀

**定义：** 一般物价水平持续上涨。

**衡量指标：**
- CPI（消费者物价指数）：消费品价格变动
- PPI（生产者物价指数）：生产者投入品价格变动
- GDP平减指数：全部最终产品和服务的价格变动

**通胀类型：**
1. **温和通胀（< 10%）：** 有利于经济增长（菜单成本小）
2. **奔腾通胀（10-100%）：** 开始有害
3. **超级通胀（> 100%）：** 经济混乱

**通胀成因：**
- **需求拉动型（Demand-Pull）：** 总需求 > 总供给
- **成本推动型（Cost-Push）：** 成本上升（工资、原材料）推动
- **货币供给过多（Money Supply）：** 过度印钱

**菲利普斯曲线（Phillips Curve）：**
- 短期：通胀率与失业率负相关（取舍关系）
- 长期：垂直的菲利普斯曲线（无取舍，取决于预期通胀）

### 2.5 宏观经济政策

#### 财政政策

**工具：** 政府支出（G）和税收（T）

**类型：**
- **扩张性财政政策：** 增加G或减税 → 刺激总需求 → 产出增加
- **紧缩性财政政策：** 减少G或增税 → 抑制总需求

**自动稳定器（Automatic Stabilizers）：**
- 累进税制（自动调节）
- 失业保险
- 社会福利

**斟酌使用的财政政策：** 政府根据经济状况主动调整

**挤出效应：** 扩张性财政政策导致利率上升，挤出私人投资。

**李嘉图等价定理（争议）：** 政府借款等价于未来税收 → 理性消费者会预存资金 → 扩张性财政政策无效。

#### 货币政策

**货币银行体系基础：**
- 货币供给 = 货币乘数 × 基础货币
- 货币乘数 = 1 / 法定存款准备金率

**央行工具：**
1. **公开市场操作（最重要）：** 买卖政府债券
2. **法定存款准备金率：** 提高→货币供给减少
3. **再贴现率：** 商业银行向央行借款的利率

**货币政策传导机制：**
```
M↑ → i↓ → I↑ → AD↑ → Y↑
```

**流动性陷阱：** 利率降到极低，LM曲线水平，货币政策失效，只有财政政策有效。

#### 货币政策规则

**泰勒规则：** i = π + r* + a(y - y*) / y*
- i：名义利率
- π：通胀率
- r*：均衡实际利率
- y：实际GDP
- y*：潜在GDP

---

### 2.6 开放经济宏观经济学

#### 汇率制度

**固定汇率：** 央行干预维持汇率稳定
- 小国盯住大国货币
- 问题：失去货币政策自主性

**浮动汇率：** 由市场供求决定
- 汇率升值：出口减少，进口增加
- 汇率贬值：出口增加，进口减少

**蒙代尔三元悖论（Impossible Trinity）：**
```
资本自由流动 + 固定汇率 + 货币政策独立性 → 三选二
```

#### 国际收支

**经常账户：** 商品和服务贸易 + 收入转移
**资本与金融账户：** 资本流动

**贸易顺差/逆差：** 出口 > 进口 = 顺差（净借入）
**资本流入/流出：** 资金净流入 = 贸易逆差融资

## 三、马斯洛心理学（需求层次理论）

### 3.1 理论背景

亚伯拉罕·马斯洛（Abraham H. Maslow，1908-1970）是美国心理学家，人本主义心理学创始人之一，被称为"第三势力心理学"的代表人物。

**理论起源：** 1943年论文《人类动机的理论》提出，后在《动机与人格》（1954）中系统阐述。

**核心思想：** 人的需求有层次之分，只有低层需求满足后，才会追求更高层次的需求。

---

### 3.2 需求层次详解（七层次）

#### 第一层：生理需求（Physiological Needs）

**定义：** 维持生存最基本的需求，是其他一切需求的基础。

**包含：**
- 食物（饥饿感）
- 水（口渴感）
- 睡眠
- 性（繁衍本能）
- 呼吸
- 体温调节
- 排泄

**特征：**
- 缺乏时，其他一切需求退居次位
- 极度饥饿的人，可以为了食物牺牲一切（甚至道德）
- 满足后，动机急剧减弱

**生活例子：**
- 饥荒中的人只求吃饱
- 贫困线以下的人"为五斗米折腰"

---

#### 第二层：安全需求（Safety Needs）

**定义：** 免于威胁、保护自己免受伤害的需求。

**包含：**
- 身体安全（免于危险、暴力）
- 经济安全（就业、储蓄、保险）
- 健康保障（医疗、养老）
- 稳定的秩序与规则
- 可预测的环境

**特征：**
- 当生理需求相对满足后，安全需求成为主导动机
- 儿童和成人的安全需求表现不同
- 成人更多表现为对未来的担忧（养老、失业）

**生活例子：**
- 生病时对医疗的强烈需求
- 购买保险的心理动因
- 地震后抢购帐篷和物资

---

#### 第三层：社交需求/爱与归属（Love and Belonging）

**定义：** 与他人建立情感联系、成为群体一员的需求。

**包含：**
- 友情（朋友关系）
- 爱情（亲密关系）
- 归属感（家庭、团队、国家）
- 被接纳、被认同

**特征：**
- 前两层需求相对满足后，社交需求成为主导
- 人类是社会性动物，孤独会带来心理和生理损害
- 社交需求比生理和安全需求更难满足

**生活例子：**
- 留守儿童的情感缺失问题
- 老年人怕孤独
- 网瘾少年在游戏中寻找归属感

---

#### 第四层：尊重需求（Esteem Needs）

**定义：** 希望自己有能力、有价值，并希望得到他人认可的需求。

**两个维度：**
1. **内部尊重（Self-respect）：** 自尊、自信、自我认可（"我对不对得起自己"）
2. **外部尊重（External esteem）：** 被认可、被尊重、有地位、有声望（"别人怎么看我"）

**包含：**
- 成就感
- 胜任感
- 能力感
- 他人的尊重和认可
- 社会地位

**特征：**
- 满足后产生胜任感和价值感
- 不满足则产生自卑和无能感
- 分为"强者"型（需要出众成就）和"弱者"型（需要他人认可）

**生活例子：**
- 职场人追求升职加薪（外部尊重）
- 学者追求学术认可
- 运动员追求奖牌

---

#### 第五层：认知需求（Cognitive Needs）

**定义：** 对知识、理解和探索的需求，属于成长需求之一。

**包含：**
- 好奇心（"这是什么"）
- 理解欲（"为什么是这样"）
- 探索世界的欲望
- 学习新事物的需求
- 理解自己和他人的需求

**特征：**
- 人类是"认知动物"，天生有求知欲
- 幼童时期表现最明显（"为什么"阶段）
- 成长需求，越满足越追求更多

**生活例子：**
- 孩子问"为什么天是蓝的"
- 科学家探索未知
- 成年人刷新闻了解世界

---

#### 第六层：审美需求（Esthetic Needs）

**定义：** 对对称、和谐、秩序、美感的需求。

**包含：**
- 对称、平衡的美感体验
- 艺术欣赏
- 音乐欣赏
- 自然美的欣赏
- 对秩序和整洁的需求

**特征：**
- 几乎所有文化中都有审美追求
- 满足后产生愉悦和满足感
- 审美缺失会导致"丑陋综合征"

**生活例子：**
- 整理房间（对称、有序带来舒适感）
- 旅行看风景
- 听音乐感动流泪

---

#### 第七层：自我实现需求（Self-Actualization）

**定义：** 实现自己全部潜能、成为自己所能成为的人的最高层次需求。

**定义：** "音乐家演奏，画家作画，诗人写诗——如果这些是他们命中注定要做的，那他们就应该做这些。"

**特征：**
- 只有极少数人能达到（马斯洛估计约2%的人）
- 每个人有不同的自我实现形式
- 是永无止境的追求
- 不是结果，而是过程

**自我实现者的特征（马斯洛的研究）：**
1. 接受现实，对现实有更准确的认知
2. 以问题为中心，而非以自我为中心
3. 自主、独立，不依赖他人的赞许
4. 超然独处，内心平静
5. 有高峰体验（peak experience）
6. 对文化有批判性洞察
7. 有创造性
8. 有深刻的人际关系

**生活例子：**
- 科学家为真理献身
- 艺术家追求完美作品
- 运动员挑战人类极限

---

### 3.3 补充：超越需求（Self-Transcendence）

马斯洛晚年提出，比自我实现更高层次：

**定义：** 超越自我，与更大的整体（宇宙、自然、他人）融为一体的体验。

**包含：**
- 神秘体验
- 宇宙意识
- 超越自我的利他行为
- 对终极意义的追求

**高峰体验（Peak Experience）：**
- 短暂的、强烈的、超越自我的体验
- 例：母亲看着新生儿、艺术家创作时的忘我状态、自然中的"天人合一"

---

### 3.4 理论评价

**贡献：**
1. 提出需求分层结构，对后世影响深远
2. 强调人的积极潜能和自我实现
3. 为人本主义心理学奠定基础
4. 在管理学、教育学、组织行为学中广泛应用

**批评：**
1. 实证研究支持不足（很难验证层次是否存在）
2. 各层次之间的界限模糊
3. 忽视了文化差异（集体主义文化中，层级可能不同）
4. "自我实现"定义模糊，难以测量
5. 忽视了"反潮流者"（不追求自我实现但也心理健康的人）

---

## 四、《竹书纪年》（古籍）

### 4.1 历史背景

#### 出土经过

**时间：** 西晋太康二年（公元281年）

**地点：** 河南省汲县（今卫辉市）战国魏墓

**情况：** 汲县人盗墓，发现大批竹简（"汲冢书"），晋武帝命官整理。

**整理者：**
- 荀勖：最先整理，定名为《竹书纪年》
- 和峤：重新编次
- 束皙：最终完成释读整理

**原始文字：** 战国古文（"蝌蚪文"），需转写为当时通行文字

---

### 4.2 两大版本系统

#### 古本（已亡佚）

**内容：** 从黄帝到战国魏襄王二十年（前299年）的编年史

**特点：**
- 原貌保存较好
- 记事与《史记》等正史多有不同
- 清代学者从类书中大量辑佚，成《古本竹书纪年辑证》

**与《史记》的主要差异：**
1. 夏朝起始：大禹禅让 vs 禹传子（纪年载"益干启位，启杀之"）
2. 商朝各王在位年数：多有不同
3. 西周积年：历来众说纷纭
4. 战国魏国纪年：比《史记》更精确

#### 今本（存世）

**内容：** 共十二篇，按夏、商、周、晋、魏纪年

**真伪之争：**
- 主流观点认为"今本"是宋代以后的伪作
- 但其中也保留了部分真古本内容
- 学者王国维作《今本竹书纪年疏证》

---

### 4.3 主要内容

#### 古本《竹书纪年》框架

```
《竹书纪年》（古本）
│
├── 黄帝至帝舜（约前3000-前2100）
│   └── 传说时代，内容有限

│   ├── 重点：商代各王在位年数（精确）

```

### 4.4 学术价值

#### 1. 纠正传世史书的错误

**"益干启位，启杀之"：**
- 《史记》等传统史籍：大禹禅让给益，益让给启，"家天下"始于夏禹
- 《竹书纪年》：益（禹的接班人）夺位，被启杀掉
- 这挑战了儒家"禅让制"的政治理想叙事

**商朝王年：**
- 《竹书纪年》提供的在位年数可与甲骨文、金文互证
- 对夏商周断代工程有重要参考价值

#### 2. 夏商周断代工程的支撑文献

"夏商周断代工程"（1996-2000年）是国家级重大研究项目，旨在确定夏商周的确切年代。

《竹书纪年》提供的：
- 夏朝积年（约471年）
- 商朝积年（约496年）
- 西周积年（约257年）
- 各王在位年数

都是断代工程的重要参考依据。

#### 3. 开创"二重证据法"

**概念：** 以出土文献（地下材料）与传世文献（纸上材料）相互印证的历史学方法。

**地位：** 王国维在《古本竹书纪年辑证》中实践这种方法，被誉为"近代学术史上第一部利用出土文献研究先秦史的典范"。

#### 4. 魏国史的珍贵史料

魏国是战国七雄之一，《竹书纪年》对魏国史事记载最为详尽：
- 魏文侯、魏武侯、魏惠王等魏国君主的事迹
- 魏国与赵、韩、秦、齐、楚等国的战争与外交
- 商鞅变法（前361年）在魏国开始等重要事件

## 五、政治经济学

### 5.1 政治经济学的对象与方法

#### 研究对象

政治经济学研究的是**生产方式**以及**生产关系**（人们在生产中形成的社会关系）。

**与西方经济学的根本区别：**

| | 马克思主义政治经济学 | 西方经济学 |
|---|---|---|
| 研究对象 | 生产方式与生产关系 | 资源配置（稀缺资源如何利用） |
| 核心问题 | 阶级关系、经济剥削 | 效率最大化 |
| 方法论 | 唯物辩证法 | 边际分析、均衡分析 |
| 价值理论 | 劳动价值论 | 主观价值论（效用论） |
| 目的 | 揭示资本主义规律，批判资本主义 | 解释资本主义运行，提出政策建议 |

#### 关键词

- **生产力：** 人类征服自然、改造自然的能力（人和物的要素）
- **生产关系：** 人们在生产中形成的相互关系（所有制、分配、交换）
- **经济基础：** 生产关系的总和
- **上层建筑：** 政治法律制度、意识形态

---

### 5.2 商品与货币

#### 商品的二因素

**使用价值：** 商品的有用性（能满足人们某种需要的属性）
**价值：** 凝结在商品中的无差别的人类劳动

**两者的关系：**
- 使用价值是价值的物质承担者
- 使用价值不同的商品，价值量无法直接比较
- 价值是商品特有的属性，是社会属性的体现

**交换价值：** 一种使用价值与另一种使用价值相交换的量的关系或比例

#### 劳动的二重性

马克思的伟大发现：

| 劳动 | 具体劳动 | 抽象劳动 |
|------|---------|---------|
| 定义 | 目的、工具、对象、结果各不同的劳动 | 撇开具体形式的一般人类劳动 |
| 性质 | 创造使用价值 | 创造价值 |
| 关系 | 具体劳动和抽象劳动是同一劳动的两个方面 | |
| 意义 | 解决了价值起源的理论难题 | |

#### 价值量与价值规律

**社会必要劳动时间：** "在现有的社会正常的生产条件下，在社会平均的劳动熟练程度和劳动强度下，制造某种使用价值所需要的劳动时间。"

**单位商品价值量：** 与生产该商品的社会必要劳动时间成正比，与劳动生产率成反比。

**价值规律（基本规律）：**
```
商品交换以价值量为基础，实行等价交换
```

**表现形式：** 价格围绕价值上下波动（受供求关系影响）

**作用：**
1. 自发调节社会资源在各部门分配
2. 刺激商品生产者改进技术、提高劳动生产率
3. 引起两极分化（优胜劣汰）

#### 货币的起源与职能

**货币的起源：** 商品交换发展到一定阶段的产物
- 简单/偶然的价值形式（1只羊 = 2把石斧）
- 扩大的价值形式（羊可以换多种商品）
- 一般价值形式（所有商品都用一种商品来表示价值）
- **货币形式**（金银固定充当一般等价物）

**货币的职能：**

1. **价值尺度：** 货币作为衡量商品价值的尺度（观念上的货币即可）
   - 价值尺度 → 价格标准

2. **流通手段：** 货币作为商品交换的媒介（必须是现实的货币）
   - 流通规律：M = P·Q / V（M=货币量，P=价格，Q=商品量，V=流通速度）

3. **贮藏手段：** 货币退出流通，作为社会财富的一般代表保存起来（足值货币）

4. **支付手段：** 货币用于偿还债务、支付工资、租金等（赊购赊销）

5. **世界货币：** 货币在国际贸易中作为一般等价物（黄金或国际储备货币）

---

### 5.3 资本与剩余价值

#### 资本总公式

**G - W - G'** （货币 → 商品 → 更多货币）

**矛盾：** 商品交换遵循等价原则，为什么G' > G？

#### 劳动力成为商品

**劳动力：** 人的劳动能力，即体力和脑力的总和。

**劳动力成为商品的条件：**
1. 劳动者有人身自由，可以支配自己的劳动力
2. 劳动者没有生产资料，只能靠出卖劳动力为生

**劳动力商品的价值：** 维持劳动者及其家属生存所必需的生活资料价值
- 包括：生活资料价值 + 培训费用 + 历史道德因素

**劳动力商品的使用价值：** 劳动（能创造价值，且能创造比自身价值更大的价值）

#### 剩余价值的生产

**剩余价值（m）：** 雇佣工人创造的、被资本家无偿占有的、超过劳动力价值的价值。

**资本主义生产过程：** 劳动过程与价值增殖过程的统一

**价值增殖过程：** 延长劳动时间（绝对剩余价值）vs 提高劳动生产率（相对剩余价值）

**两种剩余价值生产方式：**

| 类型 | 定义 | 条件 | 延长方式 |
|------|------|------|----------|
| 绝对剩余价值 | 必要劳动时间不变，延长工作日 | 必要劳动时间不变 | 延长工作日长度 |
| 相对剩余价值 | 工作日长度不变，缩短必要劳动时间 | 技术进步全社会劳动生产率提高 | 缩短必要劳动时间 |

**相对剩余价值的生产：**
- 超额剩余价值：个别企业提高劳动生产率→个别价值 < 社会价值→超额
- 所有企业都提高→全社会劳动生产率提高→相对剩余价值

#### 资本有机构成

**不变资本（c）：** 购买生产资料（原材料、机器设备）的资本 → 价值转移，不增殖

**可变资本（v）：** 购买劳动力的资本 → 价值增殖（剩余价值来源）

**资本有机构成（c:v）：** 不变资本与可变资本的比例
- 技术构成：物质关系（生产资料:劳动力）
- 价值构成：c:v
- 有机构成提高 → 利润率下降 → 资本有机构成提高是资本主义积累的一般趋势

#### 资本主义工资

**本质：** 劳动力价值或价格的转化形式（不是劳动的价值或价格）

**形式：**
- 计时工资：按劳动时间支付
- 计件工资：按产品数量支付

**工资的变动趋势：**
- 名义工资（货币工资）：表面上的工资
- 实际工资：名义工资能购买的商品和服务数量
- 相对工资：与资本家利润相比的工资

### 5.4 资本的流通过程

#### 资本循环

**三种循环形式：**
```
G - W - ...P... - W' - G'（货币资本循环）
P - ...P... - W' - G' - W - P（生产资本循环）
W' - G' - W - ...P... - W'（商品资本循环）
```

**产业资本正常循环的条件：**
1. 产业资本的三种职能形式在空间上同时并存
2. 三种循环形式在时间上相继进行

**产业资本周转：**
- 周转时间 = 生产时间 + 流通时间
- 周转次数 = 年资本价值总额 / 年预付资本总额

#### 社会总资本的再生产

**社会总产品：** 一年内社会生产的全部物质资料总和。

**两大部类：**
- 第Ⅰ部类（生产资料）：用于生产的物质资料
- 第Ⅱ部类（消费资料）：用于消费的物质资料

**简单再生产的实现条件：**
```
Ⅰ(v + m) = Ⅱ(c)
```

即：第一部类的可变资本加剩余价值，要等于第二部类的不变资本。

**扩大再生产的实现条件：**
```
Ⅰ(v + Δv + m/x) = Ⅱ(c + Δc)
```

**社会总资本再生产的核心问题：** 社会总产品的价值补偿和实物替换。

### 5.5 利润与平均利润

#### 剩余价值转化为利润

**利润（P）：** P = m（剩余价值）

当剩余价值被看作全部预付资本的产物时，就转化为利润。

**利润率（p'）：** p' = m / (c + v)

**剩余价值率（m'）：** m' = m / v（反映剥削程度）

**利润率与剩余价值率的关系：**
```
p' = m' × v / (c + v) = m' × （可变资本/全部预付资本）
```

#### 利润转化为平均利润

**竞争机制：**
- 部门内部竞争：同一部门内各企业争夺市场份额 → 形成社会价值
- 部门之间竞争：不同部门间争夺有利投资场所 → 形成平均利润

**平均利润：** 各部门资本家按等量资本获得等量利润（利润率趋于平均化）

**生产价格：** 成本价格 + 平均利润
- 价值 = c + v + m
- 生产价格 = k + P̄（P̄为平均利润）

**价值转化为生产价格：**
- 商品不再按价值出售，而是按生产价格出售
- 价格围绕生产价格上下波动
- 价值规律的形式发生了变化，但本质未变

#### 商业资本与商业利润

**商业资本：** 专门从事商品买卖的资本（产业资本的分离）

**商业利润：** 产业资本家让渡给商业资本家的部分剩余价值（通过购销差价）

**借贷资本与利息：**
- 借贷资本：货币所有者借给职能资本家使用的货币资本
- 利息：职能资本家支付给货币所有者的报酬（本质是剩余价值的转化）

#### 地租

**级差地租：**
- 级差地租Ⅰ：不同地块的肥沃程度、位置差异形成的超额利润
- 级差地租Ⅱ：同一地块连续投资形成的超额利润

**绝对地租：** 由于土地私有制的垄断，无论土地优劣都必须缴纳的地租。

### 5.6 垄断资本主义

#### 垄断的形成

**自由竞争 → 垄断：**
- 资本集中（大资本吞并小资本）
- 生产集中（生产资料、劳动力和产品集中于大企业）
- 达到一定阶段（一个部门少数大企业）→ 产生垄断

**垄断组织形式：**
- 卡特尔（联合定价）
- 辛迪加（统一销售）
- 托拉斯（合并生产）
- 康采恩（跨行业联合）

#### 垄断利润与垄断价格

**垄断利润：** 大垄断组织获得的高额利润

**垄断价格：** 垄断组织凭借垄断地位制定的价格
- 垄断高价：出售商品时高于生产价格的价格
- 垄断低价：收购原材料时低于生产价格的价格

**垄断利润来源：**
- 本国无产阶级和劳动人民创造的剩余价值
- 非垄断企业的一部分利润
- 其他国家劳动人民创造的部分价值

#### 国家垄断资本主义

**国家垄断资本主义：** 垄断资本与国家政权相结合的资本主义。

**形式：**
1. 国家与私人垄断资本在企业外部结合（国家采购、补贴）
2. 国家与私人垄断资本在企业内部结合（国家持股）
3. 国家与垄断资本完全结合（国有企业）

**国家调节：** 利用财政政策、货币政策、计划等手段干预经济。

### 5.7 社会主义市场经济

#### 基本经济制度

**公有制为主体、多种所有制经济共同发展：**
- 全民所有制（国有经济）
- 集体所有制
- 个体经济、私营经济、外资经济（补充）

**按劳分配为主体、多种分配方式并存：**
- 按劳分配：公有制经济中的分配方式
- 按生产要素分配：资本、技术、管理等参与分配

#### 市场经济与社会主义制度结合

**社会主义市场经济：** 市场经济与社会主义基本制度相结合。

**特征：**
1. 坚持公有制主体地位
2. 以共同富裕为根本目标
3. 实行有效的宏观调控

**市场机制：** 供求机制、价格机制、竞争机制

**宏观调控：** 计划、财政政策、货币政策、产业政策
