# 11-爬虫数据采集体系

## 1. 概述

团长 SaaS 的达人数据有两个来源：手动录入（达人建档）和爬虫自动采集。爬虫模块从抖音公开接口拉取达人基础信息（昵称/头像/粉丝/评分/主营类目/地域），填充到 `crawler_talent_info` 表，再由 `TalentEnrichOrchestrator` 合并到 `talent` 表。

爬虫模块与达人主表的合并链路：

```
Talent 建档（douyin_uid）
       ↓
CrawlerScheduler 定时调度（每 30 分钟 / 每 2 小时）
       ↓
DouyinTalentCrawler.crawl() → CrawlerTalentInfo
       ↓
CrawlerTalentInfoService.crawlAndSave() → upsert → crawler_talent_info 表
       ↓
TalentEnrichOrchestrator.enrich() → 合并到 talent 表
```

## 2. 爬虫基类（CrawlerBase）

所有爬虫继承 `CrawlerBase`，提供通用能力：

| 能力 | 说明 |
|---|---|
| UA 轮换 | `nextUa()` 从 `UaPool` 取一个 User-Agent |
| 请求间隔 | `sleepBetweenRequests()` 避免请求过快 |
| 指数退避 | `sleepExponentialBackoff(attempt)` 重试前等待 |
| fetch 模板方法 | 子类实现 `fetch(url)`，父类处理异常和重试逻辑 |

## 4. 达人爬虫（DouyinTalentCrawler）

### 4.1 爬取流程

```java
// DouyinTalentCrawler.java
public Optional<CrawlerTalentInfo> crawl(String talentId) {
    // 1. URL 编码 talentId，拼装目标 URL
    String url = talentUrl + "?open_id=" + encodedTalentId;

    // 2. 最多重试 3 次
    for (int attempt = 0; attempt < MAX_RETRIES; attempt++) {
        try {
            sleepBetweenRequests();           // 请求间隔
            String raw = fetch(url);          // HTTP GET
            return parse(raw, talentId);     // JSON 解析
        } catch (Exception e) {
            if (attempt < MAX_RETRIES - 1) {
                sleepExponentialBackoff(attempt + 1);  // 指数退避等待
            }
        }
    }
    return Optional.empty();  // 3 次全失败
}
```

### 4.2 fetch 实现

```java
// DouyinTalentCrawler.java:58-73
@Override
protected String fetch(String url) throws Exception {
    String ua = nextUa();  // 从 UA 池取一个
    HttpResponse response = HttpRequest.get(url)
            .header("User-Agent", ua)
            .header("Accept", "application/json")
            .timeout(10_000)  // 10 秒超时
            .execute();
    if (response.getStatus() < 200 || response.getStatus() >= 300) {
        throw new IllegalStateException("Unexpected crawler HTTP status: " + response.getStatus());
    }
    String body = response.body();
    if (body == null || body.isBlank()) {
        throw new IllegalStateException("Empty response body");
    }
    return body;
}
```

### 4.3 解析字段

```java
// DouyinTalentCrawler.java:75-93
private Optional<CrawlerTalentInfo> parse(String raw, String talentId) {
    JsonNode root = objectMapper.readTree(raw);
    JsonNode data = root.path("data");

    CrawlerTalentInfo info = new CrawlerTalentInfo();
    info.setTalentId(talentId);
    info.setNickname(textOrNull(data, "nickname"));           // 昵称
    info.setAvatarUrl(textOrNull(data, "avatar_url"));        // 头像
    info.setFansCount(longOrZero(data, "fans_count"));        // 粉丝数
    info.setCreditScore(decimalOrZero(data, "credit_score"));  // 信用评分（0~5）
    info.setMainCategory(textOrNull(data, "main_category"));  // 主营类目
    info.setRegion(textOrNull(data, "region"));               // 地域
    info.setLastCrawlTime(LocalDateTime.now());               // 爬取时间
    return Optional.of(info);
}
```

## 5. 定时调度（CrawlerScheduler）

两个定时任务，均受 `talent.data.public-page-crawl-enabled` 配置控制（默认 false，即默认关闭）：

### 5.1 每 30 分钟：新达人全量爬取

```java
@Scheduled(cron = "0 */30 * * * *")  // 每小时每 30 分钟执行一次
public void crawlTalentInfo() {
    // 从 talent 表取最近更新、最多有 douyin_uid 的 N 条达人
    List<String> targetIds = loadTargetTalentIds();  // batchSize=100
    int saved = crawlerService.crawlAndSave(targetIds);
}
```

### 5.2 每 2 小时：已有记录增量更新

```java
@Scheduled(cron = "0 0 */2 * * *")  // 每 2 小时整点执行
public void updateTalentStats() {
    // 从 crawler_talent_info 表取最久未更新的 N 条
    List<String> targetIds = crawlerTalentInfoMapper.selectList(
            .orderByAsc(CrawlerTalentInfo::getLastCrawlTime)
            .last("limit " + batchSize))
            .stream().map(CrawlerTalentInfo::getTalentId).toList();
    int saved = crawlerService.crawlAndSave(targetIds);
}
```

**分叉逻辑**：优先从 `crawler_talent_info` 查（命中则返回），查不到则回退到 `talent` 表（`searchTalentsFromTalentTable`）。

### 5.3 目标达人选择

```java
// CrawlerScheduler.java:91-103
private List<String> loadTargetTalentIds() {
    return talentMapper.selectList(
            .eq(Talent::getStatus, 1)        // 状态正常
            .eq(Talent::getDeleted, 0)       // 未删除
            .isNotNull(Talent::getDouyinUid) // 有抖音 UID
            .orderByDesc(Talent::getUpdateTime)
            .last("limit " + batchSize))      // 默认 100 条/批
            .stream().map(Talent::getDouyinUid).toList();
}
```

## 6. 数据服务（CrawlerTalentInfoService）

### 6.1 爬取并 Upsert

```java
// CrawlerTalentInfoService.java:34-48
@Transactional
public int crawlAndSave(List<String> talentIds) {
    int success = 0;
    for (String talentId : talentIds) {
        Optional<CrawlerTalentInfo> optional = crawler.crawl(talentId);
        if (optional.isPresent()) {
            CrawlerTalentInfo info = optional.get();
            mapper.upsert(info);  // 存在则更新，不存在则插入
            success++;
        }
    }
    return success;
}
```

### 6.2 搜索（样品申请选达人场景）

```java
// CrawlerTalentInfoService.java:50-84
public IPage<SampleTalentVO> searchTalents(keyword, region, minFans, maxFans, minScore, page, size) {
    // 1. 参数校验（minFans/maxFans/minScore >= 0）
    // 2. 优先查 crawler_talent_info 表（分词+过滤）
    IPage<CrawlerTalentInfo> result = mapper.searchTalents(pager, keyword, region, minFans, maxFans, minScore);
    if (result.getTotal() == 0) {
        // 3. crawler 表无数据时，回退到 talent 表
        return searchTalentsFromTalentTable(keyword, region, minFans, maxFans, page, size);
    }
    return result.convert(SampleTalentVO::new);
}
```

### 6.3 单条查询（带回退）

```java
// CrawlerTalentInfoService.java:86-111
public CrawlerTalentInfo findByTalentId(String talentId) {
    // 1. 查 crawler_talent_info 表
    CrawlerTalentInfo info = mapper.selectOne(...);
    if (info != null) return info;

    // 2. 查不到则回退到 talent 表（取基本信息构建 CrawlerTalentInfo）
    Talent talent = talentMapper.selectOne(.eq(Talent::getDouyinUid, talentId));
    if (talent == null) return null;

    CrawlerTalentInfo fallback = new CrawlerTalentInfo();
    fallback.setTalentId(talent.getDouyinUid());
    fallback.setNickname(talent.getNickname());
    fallback.setAvatarUrl(talent.getAvatarUrl());
    fallback.setFansCount(talent.getFans());
    fallback.setRegion(talent.getIpLocation());
    return fallback;
}
```

## 7. 配置参数

| 配置项 | 默认值 | 说明 |
|---|---|---|
| `douyin.crawler.talent-url` | `https://api.douyin.com/internal/talent/list` | 爬取目标 URL |
| `douyin.crawler.batch-size` | `100` | 每批爬取达人数量 |
| `talent.data.public-page-crawl-enabled` | `false` | **默认关闭**，需手动开启 |

## 8. 已知约束

| 约束 | 说明 |
|---|---|
| **默认关闭** | `public-page-crawl-enabled=false`，定时任务不执行，需手动开启 |
| **非官方 API** | 爬取目标 URL 为内部地址，非抖音公开 API，可能存在访问限制 |
| **无代理池** | 只用 UA 伪装，没有 IP 代理池，被封 IP 时无降级方案 |
| **upsert 并发** | `crawlAndSave` 逐条 upsert，批量操作无事务打包 |
