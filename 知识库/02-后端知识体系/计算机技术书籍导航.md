## 学习路线规划

### 零基础入门顺序

```
第一阶段：编程基础
  1. 《算法图解》- 算法思维建立
  2. 《JavaScript高级程序设计》- 编程语言基础
  3. 《CSS揭秘》- 前端样式技巧

第二阶段：核心知识
  4. 《你不知道的JavaScript（上卷）》- 语言核心
  5. 《深入理解计算机系统》- 系统思维
  6. 《编码》- 硬件逻辑

第三阶段：框架与工具
  7. 《Spring实战》- Java 企业开发
  8. 《Vue.js实战》或《深入React技术栈》- 前端框架
  9. 《Pro Git》- 版本控制

第四阶段：进阶深入
  10. 《Redis开发与运维》- 缓存与存储
  11. 《Linux高性能服务器编程》- 网络编程
  12. 《程序员的自我修养》- 底层编译链接
```

### Java 学习路径

```
Java语法基础 → 《Java 8 实战》 → 《Spring实战》 → 《Spring MVC+MyBatis》
                    ↓
          《Java核心技术卷2》 → JVM 深入
```

### 前端学习路径

```
HTML/CSS基础 → 《JavaScript高级程序设计》 → 《你不知道的JavaScript》
     ↓                      ↓
《CSS揭秘》           《Vue.js实战》或《深入React技术栈》
                            ↓
                   《深入理解TypeScript》
```
