# 静电场：高斯定理、电势与电场

## 概述

这一组课件把静电场复习主线串得很清楚：先用电力线和电通量直观描述电场，再用高斯定理求场强，接着引入电场力做功、电势能、电势和等势面，最后建立电势与电场强度之间的关系。

## 详细内容

### 1. 电力线与电通量

- 电力线是帮助直观理解电场方向和强弱的图像工具，不是实际存在的线。
- 线的切线方向代表该点场强方向。
- 线疏密反映场强大小。

电通量描述电场穿过某一面的总体效果，闭合曲面在这一章里统一看作高斯面。

### 2. 高斯定理

高斯定理是静电场的核心定理之一：

- 穿过任意闭合高斯面的电通量，只与曲面内包围的净电荷有关。
- 与曲面外电荷如何分布无直接关系。

这一定理反映了静电场的“有源性”，源就是电荷。

### 3. 高斯定理的使用条件

高斯定理本身总成立，但“用它直接求场强”要依赖对称性。课件重点是：

- 球对称。
- 柱对称。
- 平面对称。

典型求法步骤：

1. 先判断对称性。
2. 选取与对称性匹配的高斯面。
3. 把通量积分化简为 `E` 与面积的乘积。
4. 联立高斯定理解出 `E`。

### 4. 电场力做功、电势能与电势

静电力做功的关键特点：

- 只与始末位置有关，与路径无关。
- 因此静电力是保守力，静电场是保守场。

由此可以引入：

- 电势能：电荷和场源共同具有的能量。
- 电势差：单位正电荷从一点到另一点时电场力所做的功。
- 电势：单位正电荷从该点移到势能零点过程中电场力所做的功。

复习时要分清：

- 电势是场的性质，与检验电荷无关。
- 电势能是“电荷 + 电场”系统的性质，和检验电荷有关。

### 5. 电势叠加与常见分布的电势

课件强调电势满足叠加原理：

- 多个点电荷在某点产生的总电势，等于各自电势的代数和。

常见对象包括：

- 点电荷系。
- 圆环轴线上一点。
- 均匀带电球面或带电球体。

### 6. 等势面与电场方向

- 电势相等的点构成等势面。
- 电场线总与等势面正交。
- 电场强度方向总指向电势降低最快的方向。

这部分是把“标量电势”重新连回“矢量场强”的关键。

## 常见题型

- 用高斯定理求球面、圆柱面、无限大平面附近场强。
- 比较不同路径下电场力做功。
- 已知电势求电势差、势能变化。
- 由等势面判断电场方向和场强变化趋势。
- 场强与电势结合的综合题。
