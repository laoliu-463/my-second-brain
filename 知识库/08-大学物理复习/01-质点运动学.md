# 质点运动学

## 概述

质点运动学研究“怎么运动”，先不问力的来源，重点是用参照系、位矢、速度、加速度和轨迹来描述运动，并掌握“已知运动求速度加速度”“已知加速度和初始条件求运动”的两类基本问题。

## 详细内容

### 1. 参照系、坐标系与物理模型

- 参照系是描述运动所依赖的观察基准。
- 坐标系是参照系的数学表达，常见有直角坐标系、极坐标系、自然坐标系。
- 质点模型的核心是忽略物体形状和尺寸，只保留质量与位置，适合处理平动或尺寸远小于研究尺度的问题。

### 2. 位矢、位移、速度、加速度

- 位矢：从坐标原点指向质点位置的矢量。
- 位移：一段时间内位置矢量的增量，只看起点和终点。
- 瞬时速度：位矢对时间的一阶导数，方向沿轨迹切线。
- 瞬时加速度：速度对时间的一阶导数，描述速度大小和方向变化快慢。

在直角坐标系中可直接分量求导：

- `v = dr/dt`
- `a = dv/dt = d2r/dt2`

### 3. 曲线运动与自然坐标

- 曲线运动里，速度方向不断变化，因此加速度通常拆成切向和法向两部分。
- 切向加速度负责改变量值。
- 法向加速度负责改变方向，圆周运动时指向圆心。

对圆周运动，线量和角量要会互相转化：

- `v = rω`
- `at = rβ`
- `an = v^2/r = rω^2`

### 4. 匀变速直线运动与匀变速圆周运动

`3-第三次课2026.pdf` 主要在做这部分的小结。复习时要记住三组最常用关系：

- 匀变速直线运动：`v = v0 + at`
- 位移公式：`x = x0 + v0 t + 1/2 at^2`
- 速度位移关系：`v^2 - v0^2 = 2a(x - x0)`

匀变速圆周运动在形式上与匀变速直线运动平行：

- `ω = ω0 + βt`
- `θ = θ0 + ω0 t + 1/2 βt^2`
- `ω^2 - ω0^2 = 2β(θ - θ0)`

### 5. 运动学中的两类基本问题

第一类：已知运动方程，求轨迹、位移、速度、加速度。

- 先把 `x(t)、y(t)` 或 `r(t)` 写清。
- 求速度和加速度时对时间求导。
- 求轨迹时消去参数 `t`。

第二类：已知加速度和初始条件，求速度和运动方程。

- 本质是积分问题。
- 必须带上初始条件，否则只能得到含积分常数的一般解。

### 6. 相对运动

- 不同参照系对同一运动的描述可以不同。
- 经典力学中常用伽利略变换处理低速相对运动。
- 这一部分是后续狭义相对论的前置背景。

## 常见题型

- 已知参数方程，求轨迹方程和某时刻速度、加速度。
- 圆周运动中在线量和角量之间换算。
- 已知加速度函数 `a(t)`，结合初值积分求 `v(t)` 和 `x(t)`。
- 区分位移、路程、平均速度、平均速率、瞬时速度。
