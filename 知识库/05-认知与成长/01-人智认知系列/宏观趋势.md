# 宏观趋势

## 概述

中国正经历第二次经济转型：房地产驱动（债务居民化）→ 大宗工业品（汽车/造船/飞机）→ 第三产业。货币锚点从人口红利转向碳综合，债务转移是核心机制，经济危机的本质是穷人太稳定、富人太牢固。

## 货币锚点

| 类型 | 锚点 |
|------|------|
| 西方 | 高科技第三产业（美+石油） |
| 资源国家 | 石油/天然气 |
| 中国 | 人口红利（劳力=世界工厂） |

## 债务转移机制

> 债务 = 劳动力/未来劳动力的契约绑定
> 债务 = 炼油厂，红利 = 原油

货币锚点决定了谁承担代价。

## 三次经济转型

### 第一次：房地产（已完成）
```
驱赶农民进城 → 房子承接债务 → GDP飞速发展 → 白色家电崛起
```
- 房子是债务最优载体（无法对外卖，只能本国居民承接）
- 房地产2.0：让富人和中产接棒

### 第二次：大宗工业品（进行中）
```
汽车 + 造船 + 飞机 → 工业产能消耗
→ 工业品出海 → 债务外转
```
- 目的：债务外转，否则工业产能过剩危机席卷
- 飞机 = 最后一步（需求量最大）

### 第三次：第三产业（未来）
```
金融 + 高科技 + 知识产权 + 法律 + 医疗 + 教育 + 文旅
→ 对自然资源依赖小，债转比例极大
```

## 人口红利的商业模式

```
两个基础：
1. 对外：强大工业制造能力 → 贸易顺差
2. 对内：便捷人头税模式（城市化）
    ↓
完成人口红利盈利闭环
```

## 四种消灭贫穷的方法

| 方法 | 说明 |
|------|------|
| 消灭穷人（战争/饥饿/瘟疫） | 违反天道 |
| 刺激消费（信贷） | 寅吃卯粮，危机更猛烈 |
| 以工代赈（罗斯福） | 政府举债基建 |
| 劫富济贫（UBI） | 高税率 → 全民基本收入 |

## 消费不振的真相

> 不是老百姓有钱不敢花
> 是：收入骤减 + 贫富差距太大 = 无闲钱可花

## 政策解读原则

- 房地产：稳防崩盘（防止RMB变津巴布韦币）
- 政策方向：以稳和安全为主
- 所有刺激 = 代价传递，不直接给底层
