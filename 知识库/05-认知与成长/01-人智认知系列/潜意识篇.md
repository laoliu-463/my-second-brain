# 潜意识篇

## 概述

感觉就是潜意识，潜意识是基于漫长的生物进化过程中，面对无联系的客观环境实践作出的相对应的应激反应。人做决定都不是认知，而是潜意识。知行合一就是认知体系清晰地认知到自身的潜意识，并且助力、服从。

## 潜意识的本质

人性的贪婪、懒惰、残暴、奸诈：
- **贪婪**：获取更多生存资源的意愿
- **懒惰**：减少生存资源的消耗
- **残暴**：通过各种直接手段获得资源
- **奸诈**：面对更强个体资源掠夺的策略

以上集合是与生俱来的生存能力驱动，即潜意识——也就是平时说的"不知道为什么，反正就那样了"。

## 潜意识的运转基础

保证自身存活是基础潜意识。

**人无论多高认知和知识和社会层次，做决定的都不是认知，而是潜意识。**

从生物性解释：
- 身体面对各种情况所分泌的激素
- 同一个女人出汗，有的女人你觉得香，有的觉得臭
- 女人骂"死鬼、臭男人"也是这个原因
- 触摸女人时手心出汗或很痒，代表她的潜意识认可你

## 潜意识 vs 认知

通识教育能帮忙认知到一部分自身潜意识，然后形成唯心闭环——只在乎自己内心的想法和需求，即自身潜意识需求的一面，但会客观忽略客观反馈，形成自说自话。

### 潜意识的约束体系

| 体系 | 作用对象 |
|------|----------|
| 原有认知体系 | 拿来约束自身潜意识的 |
| 道德认知部分 | 拿来约束对方潜意识的 |
| 婚姻、私有制公有制等 | 拿来约束双方或集体潜意识的 |

**人痛苦和内耗的原因**：不知道如何正确合理对外舒展自身潜意识形成交换。

## 男女潜意识模型

- **男性潜意识模型**：控制型
- **女性潜意识模型**：资源型

举例：男人出轨容易被原谅，女人出轨不容易被原谅。

## 开智的四懂

| 层级 | 内容 |
|------|------|
| 懂过去 | 了解各种因个人认知+潜意识形成的客观事实结果，等于宏观参考 |
| 懂择人 | 了解单个个体认知体系形成方式+潜意识形成的各种对外行为输出表征 |
| 懂社会运转 | 明白个体的表征之后，明白当各种表征集合在一起的宏观表征：即经济运转和分配 |
| 懂未来 | 以个人认知+潜意识为基点，以社会运转方式为宏观表征，去预判社会的下一步行为 |

## 判断表征的简单方法

当一个人开口说"我感觉、我以为、我认为"的时候，可以简单证明他只有通识认知（只有一半，即老实人）。

**这样的人说话反着听，反着理解就是他潜意识需求，然后根据自身需要去选择是否满足或利诱或拒绝。**

## 舔狗经济崩盘

画领地的雄性自身生存存在问题的前提下，便开始只服从于生存而放弃繁衍。

自然界生物自身受到威胁或食物短缺的情况下，会选择不育甚至吃掉幼崽——古代有易子而食，现在有男女对立。

跟所谓的儒家文化圈几乎没有任何关系。
