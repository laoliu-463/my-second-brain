# 机会篇

## 概述

经济危机本质是财富再分配。中美进入"憋尿比赛"（加息vs降息），中国做三手准备：人民币绑定碳综合、一带一路债务外转、资产换绑。普通人窗口期在2024年底-2025年初，最好的资产是核心城市市中心老破小。

## 憋尿比赛

```
鹰酱（美国）：死咬不放水 → 美债收益率飙升
缅北（中国）：咬紧不放水 → 怕钱外流买美债
```

- 美国先放水：钱流向缅北购买商品，定价权旁落
- 中国先放水：钱外流买美债，给美国续命
- 双方死撑，等对方先倒下
- **谁先尿出来，对方就喝饱**

## 三手准备

### 一：人民币国际化

- 现在绑定：人口红利（劳动力）
- 未来绑定：二氧化碳（碳综合）
- 老美绑定石油 → 烧油排CO₂
- 我们绑定碳综合 → 从排放量锁死美元
- **人民币可以无限制，美元有限制，以后发多少钱我们说了算**
- 所以疯狂搞军事武器（为货币信用背书）

### 二：一带一路

```
帮穷国搞基建 → 用外汇帮他还美债
欠钱用美元计价，但用人民币还
穷国拿人民币只能找我们买东西
买家多了 → 定价权不再单一
```

- 所以建筑证书资质在房地产低迷下反而涨价（就是这个逻辑）
- 本次加息没有拉爆任何一个经济体（就是一带一路的作用）

### 三：资产换绑

- 地产2.0：许以看得见但目前拿不到的利益
- 逼迫共同参与奋斗

## 普通人应对策略

```
美元停止加息 → 上杠杆买资产
最好的资产：核心城市市中心老破小
原因：房子不值钱，值钱的是土地
```

- 螺丝战神应对：储备人性认知+社会认知，掌握一门赚钱技术（熨增模型）
- 电商直播带货到尽头了
- 下一阶段：私域电商
- **普通人25年是机会窗口期（参考毕业篇：G2格局，15年机会）**
