# 婚姻篇

## 概述

婚姻和爱情有关只是近100年的事情，以前都是父母之命媒妁之言。婚姻本质是一种劳动力、生殖、社会、经济的综合绑定所形成的契约关系，是降本增效的工具，而非爱情的产物。

## 历史背景

- 以前三妻四妾的都是豪强和地主官僚阶级
- 本质想的是生的多、资源就分化得多
- 通过联姻制度形成庞大利益集团、士族集团
- 有了王朝周期律300年（参考世家篇）
- 本身也是经济问题

## 婚姻制度的运转逻辑

### 四个步骤

```
1. 男性面临雄性竞争
   → 衡量标准：财富

2. 解放女性性自由和生育权
   → 和谁繁衍是女性自己的选择
   → 造成雄性竞争

3. 强大男性不再能者多劳
   → 只能单一选择年轻漂亮的女性
   → 造成雌性竞争

4. 灌输爱、道德、制度
   → 形成认知枷锁进行绑定
```
