# 序章

## 概述

人智认知系列的开篇。核心论点：社会真相往往与表象相反，人因认知不足而无法做出正确选择。绝大多数人的困境本质是**老实人缺乏博弈能力**，人生是乘法关系而非加法，11条线（历史/文化/教育/婚姻/工作/地产/学习/人性/经济/家庭/城市化）共同决定了命运差异。

## 人口红利 ≠ 人的红利

- 没有人提"人的红利"，只有"人口红利"
- 人口红利 ≠ 人口的红利
- 社会表象往往真相都是反的

## 三大认知开智目标

1. **懂过去**：认知是如何形成和起作用的，通过表征行为推导原生家庭和成长经历，从而快速冷读一个人
2. **懂社会**：了解个体认知 → 他人认知 → 社会运转方式
3. **懂未来**：小可预判他人预判，大可预判社会车轮

## 老实人的本质

> 老实人 = 缺少博弈能力的人

- 只知道恪守规则，只走被告知的路
- 大脑只有"因为所以"一条路
- 不敢打破规则 → 选择少 → 必须忍 → 走投无路
- 被家长/老师/老板管理 → 不会钻研/思考

## 老实人三件套

```
我弱我有理 → 我弱我抢你 → 拿来吧你 → 交个朋友 → 老乡别走
```

## 博弈的本质

> 博弈的本质 = 选择能力

- 做对选择是博弈的核心
- 老实人不做选择题 → 被动接受安排

## 人生乘法公式

```
人生 ≠ 加法
人生 = 11条线 × 认知处理运转方式 = 命运不同
```

11条线：历史、文化、教育、婚姻、工作、地产、学习、人性、经济、家庭、城市化

## 核心心法

> 预判他人预判 → 选择何人恋爱、结婚、相处
> 预判社会车轮 → 选择何工作、行业、创业方向
>
> **选择大于努力，认知决定命运**
