# 规则篇

## 概述

规则的本质是维持阶级结构。理解规则是为了从金字塔底层爬升到中间或顶层——先装满自己的酒杯。

## 货币对个人 vs 对社会的意义

| | 货币对个人 | 货币对社会 |
|--|-----------|-----------|
| 意义 | 劳动价值剩余 | 等价交换物，流通+衡量生产量 |
| 目的 | 积累到质变 | 保证大量底层存活 |

> 如果人人都有300万、年利息10万，谁去劳动？
> 劳动停止 → 社会规则崩塌 → 钱变成废纸

## 赚钱难的本质

社会设定各种困难和规则，**阻止你成为有500万的人**。500万 = 拒绝打螺丝/洗盘子/搬砖的权利 = 自由。

## 资源 vs 货币

> 钱本身没有意义，有价值的永远是资源。

- 土地是资源，矿产是资源，权利是资源，**人也是资源**
- 脑矿 = 聪明人；人矿 = 蠢人
- **富人最大的财富 = 在契约社会下拥有穷人的数量和质量**

## 婚姻经济的真相

```
结婚路径：买车 → 买房 → 装修 → 彩礼 → 嫁妆 → 婚宴 → 婚纱 → 旅游
↓ 产子 → 教育 → 20年转移支付
```

- 小城市月入5千、夫妻年入15-20万，工作30年 = 450-600万总收入
- **到老存不下什么钱**
- 房子单价 ≈ 工资的50倍，不贷款一辈子买不起

## 中产收割机制

### 进城税

- 绑定了公共资源的城市住房 = 进城税
- 房价越高 = 税负越高
- 城市中心迁移后，原购房者资源价值降低 →新区New Money接盘 → 循环往复

### 房地产的逻辑

- 城市化阶段：房子长期上涨且可交易 = 金融属性
- 去金融化后：房子 = 消费品
- **实质负利率**：有存款的人拿利息 = 背贷款的人支付

## 经济危机本质

> 经济危机的本质是**韭菜不够用了**。穷人当不了韭菜，**韭菜是中产**。

两种经济危机：
1. 生产力没有突破，增量不够，韭菜不够割
2. 原有收割模式割不动了，人学聪明了

## 阶级流动的真相

- 顶层永远不动
- 阶层流动只在底层和中产之间
- 原中产滑落底层 + 部分底层爬升中产 = 阶级流动（造富运动）
- **中产 = 最焦虑最努力的人**

## 认知升级的代价

> 个人认知在短时间内急速拔高，是一把双刃剑：
> - 获得常人难以理解的洞察能力
> - 同时失去好奇心和快乐

## 核心心法

> 从古至今，王朝更替，士族阶层从未变化。
> 理解规则 → 顺应规则 → 合理运用规则
> 目标：从最下面的酒杯，爬升到中间或最顶层，**先装满红酒**。
