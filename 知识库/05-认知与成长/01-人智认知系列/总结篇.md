# 总结篇

## 概述

一个人的房子、车子、票子是表面，核心是能力、资源、人脉和圈子。本质是认知水平，决定思维模型，从而决定一个人的内在价值和人品气质。当社会给内在价值高的人打分后，结果不会差。

## 能力与认知的关系

能力、资源、人脉、圈子的本质：
- **能力**：对内认知越狱，对外实践验证
- **资源**：认知兑换的媒介
- **人脉**：能力的外延
- **圈子**：同类认知的聚集

**认知越狱**：大量具体表现形式 → 向上提炼抽象到人的本质 → 总结规律 → 向下兼容 → 知行合一

这个过程不是一次性的，而是次次性的。

## 改命的路径

**改命即具象—抽离—总结—能力代继—帮助他人**

- 帮助的人群多少决定了市场大小
- 帮助个体困难的大小，决定了单位收益的多少
- 帮助不同属性的个体，决定了收益难度

**开智的四个阶段**：崇拜 → 信仰 → 攻击 → 自负（跳跃过去就毕业了）

## 自律的本质

自律的人容易成功：
- 从人性角度：巨大欲望压制小欲望
- 从外在角度：把无序且失控的东西整理、抛弃，变得可控

**核心逻辑**：激发贪婪 → 控制自身欲望的释放 → 用大欲望压制小欲望 → 内心稳定、有序、可控 → 专注实现目的

## 相对权力

| 时代 | 相对权力 |
|------|----------|
| 当年改开 | 经营权 |
| 电商时代 | 展示权 |
| 短视频/网红时代 | 话语权 |
| 下一个时代 | AI + AGI |

**核心规律**：通过信息差 → 认知差 → 生产效率差 → 资产差

老实人开局输，一辈子没机会。因为只喜欢看自己想看的，听自己想听的，在信息差这一步就输了。

## 认知模型分类

人生必经阶段：
1. 盲目自信的愚昧之巅（巨婴傻逼）
2. 绝望无助的深渊（普通人）
3. 觉醒开悟（智者）
4. 内心与外界回馈的持续平衡（古井无波）

**人生比的是**：谁更快爬出二级闭环的坑，坑越小爬得越快，起步越快。

- 狗 = 认知底下的老实人
- 男权即生产力
- 女权是机遇生育力和养育力的消费力
