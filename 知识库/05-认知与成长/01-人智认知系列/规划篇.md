# 规划篇

## 概述

改变命运只有四个路径：结婚、参公、创业、知识。核心是找准自身定位，选择最优路径。

## 定位原则

> 现实没有大卫战巨人的故事，全是龙找龙、虾找虾，乌龟绿豆配王八。

- 天生我材必有用 ≠ 人人皆可成王侯将相（后者是被放大化的小概率事件）
- 综合最优解：**术业有专攻**
- 定位不准：好高骛远 / 大材小用

## 改变命运的四个路径

### 1. 结婚

- 婚姻 = 人生第一契约，不只是结婚证
- 软饭硬吃的故事，灰姑娘的现实版

### 2. 参公（考公）

- 目的：**进入社会管理/利益分配阶层**
- 不是为了金饭碗，而是获得分配权
- 参考：《黑冰》《素书》

### 3. 考证（职业技能证书）

#### 为什么考证 = 资产？

- 唯一投入产出成正比的路径
- 证书 = 参与社会二次分配的凭证 = 城里人的土地

| 证书 | 等效房产价值 |
|------|-------------|
| 环评工程师 | 500万（年收18万） |
| 一建 | 190-370万 |
| 注册安全工程师 | 220-350万 |

- 考证 = 被动收入 = 护身符 = 免死金牌
- **人才落户**：证书可解决户口，节省数百万购房成本

### 4. 经商

#### 三个层次

1. **生产交易**：养猪→卖猪（养殖户）
2. **资源整合**：猪贩子→屠宰加工→零售
3. **资源垄断**：种猪培育和检疫

#### 平台经济的本质

- 百度、美团、抖音都不是商品生产者
- 只做了**资源整合**这一件事

## 去中心化：普通人的25年机会窗口

- 以前：村口大妈 + 新闻联播 = 信息渠道
- 现在：人人可发表、人人可卖商品
- 过程：去中心化 → 优胜劣汰 → IP集中 = 普通人的机会
- **时间窗口：约25年**

## 宏观方向

```
40年人口红利 → 房地产居民负债化 → 产业化工业资本
→ 全产业链工业化 → 船舶飞机工业
→ 一带一路 + 巴黎协定（碳综合）+ 亚投行
→ 特高压 + 高铁 + 新能源（光伏+电车+储能）
```

本质：**路权 vs 海权的斗争**（谁的运输成本低，谁就获得老大资格）

## 核心心法

> 命由心造，运由势成。

- 以结果/目标推导和修正过程，不以过程计算结果
- 变量是未知的，有明确目标就会不自觉朝目标走
- **人的目标应该是星辰大海**
