# 地产篇

## 概述

房地产是国家利益循环的工具，本质是通过一次次资产锁定完成对不同群体的"收割"。核心是**土地的流动和利益分配**。

## 土地价值本质

> 土地的价值由这片土地上所承载的人的**单位时间生产率**决定。

- 人多不是关键，人才才是关键
- 每个城市的宣传（成都熊猫、淄博烧烤等）本质都是**抢人**

## 房地产运作规律

### 老破小定价逻辑

- 由**租金**决定，而非地段
- 学区房等是虚假价值，可随时撤离

### CT 资产公司操作

1. 冒充刚需一套一套购买老破小
2. 统一租房市场（有学位、医疗、地段）
3. 砸盘其他老破小定价
4. 完成**控盘** → 拿到城市核心地段土地

### 房票制度

1. 连拆迁款都不想给，打白条
2. 一年内需买指定拆迁房
3. 多次试探底线，逼迫上车
4. 最终完成资产锁定

## 利益链条

```
螺丝仔（租房）→ 消费 → 老破小业主出手 → 改善住房锁定 → 城市近郊别墅
```

## 未来预判

### 34 大中城市名单内

- 老破小可赶紧上车
- 第一波吃螃蟹的人会获得奖励

### 34 大中城市名单外

- 大概率没有机会

### 应对策略

1. 关注统租房/CT 公司入股机会
2. 考取相关资质（注册环评工程师）
3. 理解庄家目的，提前选择站位

## 核心结论

> 任何不能流通、交换的东西都没有意义。
> 房票的流通肯定会放开。
