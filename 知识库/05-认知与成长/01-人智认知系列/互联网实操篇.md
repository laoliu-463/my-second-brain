# 互联网实操篇

## 概述

短视频进入存量时代 = 网红阶层固化。普通人做互联网的核心不是模仿大网红，而是找到那些被特定困境困住的精准人群，通过内容让他们找到你。变现的关键是找对人群，而不是堆粉丝量。

## 普通人做互联网的前提判断

**你有供应链渠道吗？**

- 没有供应链 → 不要做直播带货
- 赚钱的是供应链，不是主播
- 辛辛苦苦搞流量 → 收益和劳动成本完全不成正比
- **不要做**：无货源带货、无粉丝变现（都是死路）

## 大网红变现路径剖析

### 小杨哥路线

```
家庭搞笑短视频 → 直播打赏 → 接广告
        ↓
粉丝量突破量级
        ↓
直播带货（目标：下沉市场，贪便宜心态）
```

### 大蓝路线

```
讲述几次创业经历 → 通过抖音翻身
        ↓
利用产生同竞的人群
        ↓
卖商业思维课（线上+线下酒会）
        ↓
有原有产业的人能用到 → 其他人成为韭菜
```

### 敬翔路线

```
厚黑学话术 + 明星包装
        ↓
按奢侈品路线包装商业课程
        ↓
利用宗教信仰植入
        ↓
客户学不懂 → 自责 → 不怪他讲不明白
```

### 财经老莫路线

```
不断爆料黑暗规则和暗箱操作
        ↓
给粉丝涨智慧涨见识
        ↓
获得粉丝信任
        ↓
金融居间（资金过桥、拆借需求的客户）
        ↓
把自己塑造成黄金圣斗士
```

## 普通人应瞄准的路径

### 核心问题：人在什么时候会毫无保留地付出一切？

- 追求女生，爱的死去活来
- 搞定客户，搞定了就升官发财
- 生病了，到处求医无门
- 买限量款汽车，全网买不到

### 答案

**如何让这些人群找到你，才是我们应该做的**

→ 粉丝量不一定要大（不超过50万），但变现能力可以比肩几百万粉丝的搞笑博主

> 短视频进入存量时代 = 网红阶层固化。除非有特殊运气和碰上特殊事件，大网红路线是死路

## 执行原则

- 不要好高骛远（别人赚100万，你目标10万就行）
- 做转化需要大量实操经验和行业专业知识储备
- 先对标、抄袭，再慢慢找到自己的路
