# 思维模式篇

## 概述

人的认知不公平是最大的不公平。四大思维模式决定了人对世界的理解方式：小农思维（等天意）→ 穷人思维（越努力越幸运）→ 富人思维（先评估再行动）→ 强者思维（多因多果来回验证）。大部分人的悲剧在于思维模型的局限导致选择错误，而非努力程度不够。

## 小农思维

**特征**：春天种地，秋天等待收获。风调雨顺还是严寒酷暑，全凭老天。把自己的劳动交给别人分配，天生把自己定位为**被管理者**。

本质：**只管努力，剩下的交给天意**。古代皇帝自称"天子"也是这个逻辑——谁是他们的天，谁就可以为所欲为。

当他们的"天"没有给予满意回馈时，只会感叹一声：**"这都是命"**。

## 穷人思维

穷人思维本质是**小农思维的升级版**：
- 前半部分没变：只管努力
- 后半部分改变：从交给天意 → 越努力越幸运，命运在自己手中

**致命弱点**：一旦努力后没有取得理想结果，第一反应是"是不是自己做得不够好"，然后加大努力程度、加大付出——**自我CPU**。

在感情中：把自己设定为供养者角色，别人对自己不满意，一定是自己付出得不够。**从始至终没有思考过反抗**。

## 富人思维

富人思维 = 穷人思维的**倒用 + 选择判断**。在付出劳动之前，先衡量付出和收益的比值，而不是盲目先做。

**优势**：一定程度避免劳动浪费，先评估风险。

**常见大坑**：人的贪婪使其选择投入产出比最高的选项，忽略外部因素和自身能力上限。常见：资金链断裂、杠杆太大扯着蛋。

## 强者思维

**核心区别**：单线思维 vs 多线思维。

| 思维类型 | 路径 |
|----------|------|
| 小农思维 | 从因到果（交给天意） |
| 穷人思维 | 从因到我果（努力=收获） |
| 富人思维 | 从我果到因（先评估再行动） |
| **强者思维** | **多因到多果 + 多果到多因来回验证** |

**强者思维实战示例——自卑溯源**：
```
老实人觉得自己自卑 → 原因：没本事，家里穷
  ↓ 倒推一次
家里穷 → 父母也没本事
  ↓ 继续
父母穷 → 认知和能力不到位
  ↓ 继续
家里没有强有力的一家之主
  ↓
母强父弱 → 母慌则闹 → 母闹则子不安
  ↓
长期不安 → 用沉默保护自己 → 自卑
```

一个原因可以造成多种结果，一个结果也可能是多种原因导致。强者思维能透过事物的呈现状态，看出背后运转规律。

**强者思维为何不容易失败**：
- 富人思维：选投入最小、回报最大的路径
- 强者思维：选投入相对较小、产出相对较大、不一定最大但**最可靠**的路径
- 核心衡量：**不看产出，看投入和产出的比值**
- 第一要素：**存活**，而不是盈利
- 哪怕失败，损失都可以控制、承受，或转嫁

## 核心结论

> 人与人最大的不公平，不是出身、学历、家庭、人脉，而是**认知上的不公平**。

因为人生可以凭运气蒙对很多次，但只要输一次，就重来。

**思维提升的代价**：管理难度和控制成本的提升。这个也是为什么漫长学习生涯中，没人给你提升思维的原因。
