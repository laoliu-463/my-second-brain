# 世家篇

## 概述

世家 = 世世代代掌握资源的家族 = 社会管理阶层。普通人难以跨越阶级的根本原因不是缺钱，而是无家族力量。世家通过血脉纽带形成利益共同体，以贡献换取话语权→管理权→分配权。

## 世家的形成逻辑

```
皇权不下县，县里靠乡绅
    ↓
世家 = 门阀 + 士族
    ↓
历代打击世家（力量太大，与降本增效背道而驰）
    ↓
现代：城市化 = 契约化 = 打断血契形成条件
```

## 世家对外核心：对冲，风险均摊

- 诸葛家三兄弟（诸葛亮、诸葛诞等）分别在魏蜀吴三国任职
- 无论谁赢，诸葛家都是赢家
- 联姻制度形成政商结合（广东福建模式）

## 世家内部管理核心：利益共同体

- 血脉为纽带，贡献决定话语权
- 话语权 → 管理权 → 分配权（分配血亲内所有人的劳动价值剩余）
- 能力强的孩子集中资源，其他孩子辅助
- **不争私欲，争家族管理权和分配权**

## 普通人 vs 世家的对比

| | 穷人思维 | 世家思维 |
|--|---------|---------|
| 资源分配 | 搞平均主义，分散有限资源 | 集中资源给最强后代 |
| 兄弟关系 | 付出多的一半有能，付出少的得更多 → 不和 | 以贡献换话语权，家族合力 |
| 代际传承 | 穷 → 穷 | 强 → 强 |

## 历代打断世家形成的方式

### 1. 城市化（契约化）
- 避免血契存在的空间
- 打工、外卖、家政、物业 → 一切经济契约
- 无佣人、无土地 → 无法形成世家基础

### 2. 计划生育
- 农村有土地 = 有资源 = 代代生娃 = 底层世家基础
- 计划生育打断这个过程
- 大学毕业生直接进入管理阶层 → 加速利益集团形成 → 再次打乱

### 3. 婚姻制度重构
- 西方爱情绑定婚姻 = 近100年的事
- 男女平等 + 一夫一妻 = 物理性打断家族形成初始条件
- 给女性灌输消费主义，给男性灌输顺从照顾 → 大量劳动剩余价值转移

## 现代世家的代理机制

- 顶层世家 → 推出代理人 → 话事人
- 目的：
  1. 树立信仰，激发虫儿中的聪明虫努力
  2. 方便背后控制，出事有人背锅

## 普通人应对策略

> 没有家族力量，就靠认知抱团
> 学会利益共同体思维，不是单打独斗

- 识别世家运作逻辑，顺应规则
- 通过考证/参公等方式进入分配阶层
