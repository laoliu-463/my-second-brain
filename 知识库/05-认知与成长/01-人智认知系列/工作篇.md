# 工作篇

## 概述

打工≠工作。先打工，再工作，才是逆袭的唯一出路。核心是**获得话语权和分配权**。

## 打工的本质

> 工作收益的本质是雇佣者对被雇佣者的一份**承诺**而已。

- 赚大钱了，可以不分你
- 账本增效了，必定辞退你
- **主动权和解释权从来不在你手里**

## 公司运作真相

- 公司分配权：老板有绝对的主动权和分配权
- 打工人越努力，老板越富裕
- 掌握资源的人规定：资源收益必须大于劳动收益

## 核心认知

### 1. 做主才是根本

> 别人答应你的都不算数，只有你自己能做主的才算数。

- 能做主的事情多一点，过得好的概率就大一点
- 要守的规矩少一点，能赚的钱就多一点

### 2. 打工是阶段性策略

- 维持基本生存阶段，打工是唯一出路
- 但必须是**阶段性**的，不能是永久性
- 打工不给人思考时间，聪明的员工会增大管理成本

### 3. 如何定规矩

> 答案：**能力**

- 组成集体后，因能力差异形成管理与被管理
- 能力高低由认知水平决定
- 先打工，再工作，才能获得话语权和分配权

## 逆袭路径

```
打工（积累）→ 掌握社会实际知识 → 获得分配权 → 定规矩
```

## 警示

- 如果父母是社会底层，人生的重大选择不要听取他们的建议
- 摆脱被剥削状态是跨越阶层的第一步
- 越努力越不幸 → 永远逃不脱贫穷陷阱
