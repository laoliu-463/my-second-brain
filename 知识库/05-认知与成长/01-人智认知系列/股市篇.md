# 股市篇

## 概述

A股本质是**承担政府职能的转移支付市场**，而非价值投资市场。全世界的国民管理基本方式是：让大部分人没有积蓄，阻止原始资本积累。

## 国际金融战格局

### 美国金融战路数

1. 拉爆香港离岸市场 → 传导在岸市场（大A + 楼市）
2. 拼命做多日本、印度 → 釜底抽薪国内流动性
3. 引发本土资金大规模外逃
4. **套路与97年东南亚危机完全一致**

### 中国的应对

- 战略定力：不大规模放水
- 定向放水：碳积分 + 专项债 → 失去土地财政但有自然资源的地方政府
- 只要熬过节点 → 货币端刺激加码 → 房屋流动性缓解
- 关键战场：**香港离岸市场**（中美直接白刃战）

### 判断标准

- 人民币汇率从 7.15-7.2 区间稳定到 7.1-7.15 → 可尝试进场
- 目前（约2024年）：约7.18，尚未进入上升通道
- 熊市/空头行情未走完

## A股本质

### 与美股对比

| | A股 | 美股 |
|--|-----|------|
| 功能 | 政府转移支付 + 国家投资 | 价值投资 |
| 优质企业 | 茅台等少数 | 大量 |
| 腾讯/京东/阿里 | 都不在A股上市 | — |

### 为什么优质企业不进A股？

- 监管环境、圈钱套现特性
- 犯罪成本低
- A股 = 赌场，不适合价值投资

### A股不存在长期牛市的基础

- 历次牛市短期上扬不超过24个月 → 漫长熊市
- 不是谁决定的，是过去30年统计结果

## 散户被收割机制

```
基金公司发行基金 → 基民高位接盘
→ 基民钱买股票
→ 基金公司股票质押给机构
→ 机构甩卖做空 → 股价下跌
→ 低价购回还基金公司
```

**基金公司赚两回钱：**
1. 基民被套住的管理费
2. 借给机构的息差利润

## 时间预判

- 行情在绝望中产生，在犹豫中生长，在欢乐中死亡
- 25年第二到第三季度，才是春天
- 目前：多头不死，空头不止
