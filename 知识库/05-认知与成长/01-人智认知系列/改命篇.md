# 改命篇

## 概述

改变命运的四个路径（结婚/参公/考证/创业）是方法论，核心心法是：乱世（危机）= 阶梯，混乱是阶级跃升的机会。互联网是普通人弯道超车的工具，关键在于追求"干"（干货）而非追求流量本身。

## 核心警示

> 老实人千万不能去追求稳定的秩序
> 经济学：稳定性越高，收益性越低
> 社会学：稳定性越高 = 被剥削的位置 = 打螺丝

## 危机即阶梯

> 认知的提高带来自信和胆量
> 经济危机 = 老实人的机会
> 真相：**混乱即是阶梯，阶级的阶梯**

- 管理者利益来源于预期一致
- 危机打破预期 = 打破固化秩序
- 牧羊人只有一人，羊乱跑时饿狼才有机会

## 互联网的本质

```
比的是认知和能力
有钱/有权都不好使
有特定才华才好使
    ↓
有话语权 → 管理权 → 分配权
```

## 改命六式

### 1. 成为是体验的上游，行为的下游

- 想成为有钱人 → 按有钱人的方式行动
- 负债时也在按有钱人的方式行动 → 本质一样

### 2. 永远肯定自己

- 人天生不接受被否定
- 早起创业骂员工，现在只表扬

### 3. 除了生理极限，其他都不是极限

- 人只能做到自己理解并相信的地步
- 自我局限 = 精神上的自我矮化
- 欲望放大过程：赚100万是人生赢家 → 真赚100万发现不过如此

### 4. 早上布局，晚上复盘，周而复始

- 顺序错了 = 晚上雄心壮志，白天一脸懵逼

### 5. 反自我人性，顺他人性，满足他人性

- 发视频说大龄剩女要价高 → 疯狂点赞
- 说老实人自身毛病 → 没啥流量

### 6. 先认命再改命

- 不抬杠，客观事实成立就听
- 老实人总觉得自己比无正确 = 龙傲天

## 繁衍主线逻辑

> 人一切追求 = 繁衍 = 性
> 繁衍快乐 = 可以跳过繁衍直接满足快乐

- 搞笑/逗比/卖惨 → 获得同情（比你惨 → 潜在快乐）
- 人的幸福和痛苦都来自对比

## 互联网3.0 + 商业6.0 时代

1. 做互联网，边做边改
2. 直到符合客观反馈
3. 先抱怨 → 发现共性问题 → 1个月涨粉6万
4. 发现老实人 = 认知体系问题 → 着手开智
