# 互联网商业逻辑

## 概述

互联网商业的核心是**同竞**与**信念植入**。商品价值 × 信念 = 商品价格，信念植入的广度决定品牌价值，深度决定品牌溢价。理解了这两点，就理解了绝大多数商业模式的底层逻辑。

## 同竞行为制造

**同竞 = 同类之间产生的对立或对标竞争**。

同竞的产生来源：欲望、焦虑、希望、胜负心。典型表现："别人可以，那我也行"。"他能成功，为什么我不行？"

本质是利用了人从小被灌输的概念：**天生我材必有用，王侯将相宁有种乎**。这是人性"嫌你穷怕你富"的典型体现。

**同竞=门槛**：差距可以追平才能产生同竞。白手起家、创业失败、重建普通——这些背景让人与博主产生同类感，哪怕现实差距已追不上，仍会产生同竞。但如果对方是富二代、权二代，则产生不了同竞，只有羡慕或嫌弃。

**如何制造同竞**：展示与目标人群基础属性相同的经历，降低初始门槛，制造"差距可以追平"的感知。

## 信念植入逻辑

**信念 = 植入概念，使被植入者深刻认同，并能够自证其说，形成群体行为**。

**商品领域**：买LV、Gucci花几万不心疼，普通塑料包几百块却嫌贵。本质都是包——但被植入"奢侈品就应该贵"的信念后，就无脑相信。

**奢侈品模式**：
```
明星/名人使用 → 第一波富人进场 → 中产模仿 → 形成集体行为 → 穷人省吃俭用购买 → 富人出手限量款离场
```

**资本市场圈钱模式**：
```
大佬站台 → 画PPT大饼 → 第一波韭菜进场赚钱 → 四处炫耀 → 后续韭菜进场 → 完成收割
```

两种模式的底层逻辑完全一致。

**为什么要用厚黑学话术包装**：人在面对未知或未解事物时，第一反应是敬畏和害怕，然后观察并试着理解；若理解不了，立刻退回道德判断路径——"我不懂，大家都不懂，就是牛逼"，完成**价值认同，即信念植入**。

## 核心公式

```
商品价值 × 信念 = 商品价格
```

- 信念植入的**广度** = 品牌价值
- 信念植入的**深度** = 品牌溢价
- 最极致的信念植入：宗教（这辈子信教，下辈子人上人）

## 表面逻辑与底层逻辑

商业类博主用厚黑学话术，是**包装**而非本质：
- "人都是贪恋的、懒惰的，需要一个合理偷懒的借口" → 包装后的说法
- 实际就是：通过植入信念的商业模式，信念植入深度 = 转化率，信念植入强度 = 利润率

**包装的目的**：利用人性弱点——人在面对未知事物时，不懂 → 敬畏 → 退回到道德判断 → 不懂装懂 → 大家都说牛逼 = 真的牛逼 = 完成了价值认同。

## 商业落地执行

**如何制造同竞行为**：
- 同 = 同类，即差距不大
- 门槛设计：差距必须可以追平
- 案例：展示与受众相同的基础属性（同样出身、同样失败经历）

**如何植入信念**：
- 传授逻辑思维或传授事物结果
- 在实践中自圆其说或使受众获得可观回馈
- 案例：小时候"努力 = 好成绩"，奢侈品"贵 = 品质"，都是完整的信念植入过程
