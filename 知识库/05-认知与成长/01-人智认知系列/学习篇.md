# 学习篇

## 概述

读书改变命运的时代已过。受教育和学习完全是两回事：受教育获得劳动资格，学习获得洞悉社会规则的能力。

## "读书改变命运"的真相

### 古代

- 读书 = 考取功名 = 进入仕途 = 成为管理者
- 读书是**唯一**从劳动阶层进入士族（管理阶层）的合法途径
- 目的：对商人地主进行资源掠夺

### 现代

- 读书 = 获得脑力劳动能力 = 多一个选择
- **最后一批靠读书改变命运的人：80年代**
- 之后学历贬值——管理和工作岗位分完了，不包分配

## 受教育 vs 学习

| | 受教育 | 学习 |
|--|--------|------|
| 目的 | 获得脑力劳动资格（白领） | 洞悉社会和人性的底层逻辑 |
| 本质 | 技术工人培养 | 资源掌控者培养 |
| 代表 | 普鲁士教育 | 王道教育（世家教诲） |

### 三种教育模式

1. **儒家教育 / 普鲁士教育**（99.9%的人接触到的）
2. **王道教育**（高知识分子接触到管理学/商官学/心理学）
3. **世家教诲**（二代们刚出道就懂社会运转规则，比普通人早 5-10 年）

## 成长过程中的认知陷阱

### 服从性训练

- 听话 = 有地位 = 有回报
- 设立小队长、班长等职务，强化"听话=正向反馈"

### 收益确定性训练

- 考试分数 = 排名
- 付出 = 回报，努力 = 成果
- **根深蒂固但到社会上完全失效**

### 社会现实

- 收益不再可见、不再确定
- 用原有逻辑生存 → 巨大自我怀疑、迷茫、痛苦
- 鲁迅笔下的骆驼祥子，到死都觉得是自己不够努力

## 核心认知

> 受教育是为了获得高效劳动或者脑力劳动的能力；学习是为了获得**看清事物运行底层逻辑**的能力。

- 成功学 = 满足底层人对上层人思维的幻想 = 精神鸦片
- 各大微商、会销的底层逻辑：制造幻想 → 收割
- 混得好的人两类：调皮捣蛋（提前认知社会规则）、真学霸（获得高阶分工入场券）

## 打工的真相

- 打工 = 内心接受被剥削地位
- 工资 = 认可窝囊安慰费机制
- 应该接受：**合作 + 利益分配 = 共赢**

## 觉醒路径

```
学校教育（服从性+确定性）
    ↓
推翻原有认知体系
    ↓
建立客观判断 + 独立思考
    ↓
知己知彼 → 做出正确方向选择
```
