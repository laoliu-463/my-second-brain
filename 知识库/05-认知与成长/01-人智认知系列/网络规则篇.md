# 网络规则篇

## 概述

在互联网上做事，必须先理解平台规则，再面对客户。不同平台的审核机制和权重分配流程各不相同，内容制作的核心逻辑是：反转→冲突→形成观点和立场的对立，利用用户"屁股决定脑袋"的客观情况。

## 先规则，后客户

大多数人的错误：一上来就面向客户，忽略了平台规则。

**正确顺序**：
```
了解平台规则 → 满足平台条件 → 面向客户输出内容
```

**平台规则速览**：

| 平台 | 核心规则 |
|------|---------|
| 百度 | SEO推荐机制 |
| 贴吧 | md7审核 + 账号权重 |
| 小红书 | 机器+人工检测 |
| 抖音/快手 | 标签权重 + 用户行为数据反馈 |
| 知乎 | 问答点赞 + SEO模式 |

## 用户行为背后的需求逻辑

| 行为 | 背后需求 |
|------|---------|
| 点赞 | 认同 |
| 评论 | 有争执 |
| 转发 | 说中了理想对外需求 |
| 收藏 | 解决了长期困扰的问题 |
| 关注 | 认可为同类和信仰对象 |

## 短视频内容制作逻辑

### 第一步：精准画像

确定用户画像 + 自身标签定位（一旦确定，后续转行损失大量客户）

### 第二步：内容核心逻辑

```
反转 → 冲突 → 再反转 → 再冲突
        ↓
  引起目标受众情感波动（终峰定律）
        ↓
  观点顺从大多数，打击小部分
        ↓
  形成对立的逻辑局面/利益对立局面
```

### 第三步：精准对接

通过视频更新迭代，精准画像目标客户，完成精准对接，最后形成交易

### 变现路径示例（建工类）

```
工地搞笑视频 → 职称证书收入/待遇差别 → 考证视频
        ↓
找到建筑行业想考证的人
        ↓
对接教育培训机构
        ↓
完成交易
```

## 优质内容的核心要素

**不一定需要高质量，但一定要能引起**：
- 情感共鸣
- 观点争执
- 信仰膜拜
- 需求对接

**最有效**：引起争执和共鸣

### 流量话题

- 男女对立
- 不买房
- 躺平
- 00后整顿职场

## 核心原则

> 你在人家的地盘上找吃的，就必须遵守或玩转别人的规则
> 二进制的代码可不知道什么人情事故
