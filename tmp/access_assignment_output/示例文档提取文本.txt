Access数据库查询编写流程
将实验素材“Access素材(事业单位人员信息表).xls”的内容导入ACCESS数据库中，并建立查询窗体，实现根据不同的分类自动查询。
下面会讲解两种不同的方法，请自行选择适合的方法进行学习，并完成作业。
一、准备工作
1、从QQ群文件中下载“Access素材(事业单位人员信息表).xls”
2、从QQ群文件中下载“Access作业-机自250X-张某某.accdb”
3、确保两个文件保存在同一个目录中，方便调用
二、安装Access
1、安装Access。Access是微软office中的一个组件，所以要使用Access就必须要安装office2016或更高版本，建议安装2016版本。因为新版本中对Access做了较大的调整，原有的设置方法可能会出现错误，也不利于文件的传递，别人可能打不开。
三、导入数据
3.1、安装完成Access后。打开下载好的文件“Access作业-机自250X-张某某.accdb”，并修改班级和姓名
3.2、打开“Access作业-机自250X-张某某.accdb”在外部数据选项卡中点击“新数据源”→“从文件”→“Excel”,选择“Access素材(事业单位人员信息表).xls”，点击确定进入导入数据表向导。第一页直接选下一步，第二页要选中第一行包含列标题，第三页直接选下一步，第四页选我自己选择主键。选择序号。点击下一步。导入到表命名为“人员信息表”。点击完成按键，完成Access数据的导入。
注：不包括双引号“”，此后若非特殊说明，均不包括双引号。
3.3、右键点击人员信息表，选择设计视图。在设计界面中，选择出生年月。
3.4、在下方的设置列表的常规选项卡中，将格式设置为“yyyy\年mm\月dd\日”。保存并关闭设置界面
四、创建查询窗体
1.1、在创建选项卡里找到空白窗体选项，点击建立空白窗体。点击保存，窗体名改为“人员信息查询”。
1.2、在设计选项卡里选择“”标签，用鼠标在空白窗体的的界面上拉出矩形窗口。写入“建邺市活动人员信息统计表”，并在开始选项卡中将字体设置为“黑体”，字号为“24”
第一种，全部使用列表框进行筛选查询（代码多）
一、建立查询
1.1、创建地区表列表：在设计选项卡中选择“”列表框控件，在窗体的空白处拉出矩形窗口。
1.1.1 在弹窗的界面中选择“使用列表框获取其他表或查询中的值”，选择下一步继续。
1.1.2 视图里选择“表”，选择框里选“表:人员信息表”,选下一步继续。
1.1.3 在可用字段(A)里鼠标双击 “区名”，或选中后点击“>”，将选择的值移动到右方选定字段(S)里。点下一步继续
1.1.4 顺序选择不要做任何设置，点下一步继续。在弹窗对话框中不要做任何改动，下一步继续。
1.1.5 在“请为列表框指定标签”下的对话框里录入“地区表”。完成列表框创建。
1.1.6 修改列表框设置（如果不进行调整设置，列表框内的数据会非常杂乱）。如果界面右侧没有属性表设置界面，可以使用鼠标右键点击创建好的“地区表”列表框，选择属性，打开右侧的属性表。点击属性表的其他选项卡。将名称改为“地区表”
1.1.1 在属性窗口的数据选项卡中，找到行来源，点击右边的“…”打开编辑界面。将下方的字段设置里的“序号”列删除。
鼠标右键菜单中 项卡里找到。总计选项里选择“Group By”，关闭并保存地区表查询编辑窗口。
1.1.9 在属性表的格式选项卡里，列数选择“1”，列宽设置为“2.542cm”，列标题选“是”。在属性表的数据选项卡的里，绑定列选1。
1.2、创建单位表列表：同样的方法创建新的列表框，视图里选择“表”选择框里选“表:人员信息表”。
1.2.1 将可用字段(A)里的“区名”、“单位”，按顺序移动到选定字段(S)。在请为查询指定标题:的对话框里录入“单位表”，点击确定完成创建。
1.2.2 到属性表的数据选项卡中的行来源，点击右侧的“…”打开编辑界面，删除“序号”列，点击、保存并关闭单位表查询编辑窗口。
1.2.3 点击属性表窗口其他选项卡。将名称改为“单位表”。
1.2.4 在属性表的格式选项卡里，列数选择“2”，列宽设置为“0cm;2.542cm”，讲列标题设置为“是”。在属性表的数据选项卡的里，绑定列选:“2”。
1.3、创建科室表列表：同样的方法创建新的列表框，视图里选择“表”选择框里选“表:人员信息表”。
1.3.1 将可用字段(A)里的“区名”、“单位”、“科室”，按顺序移动到选定字段(S)。在请为查询指定标题:的对话框里录入“科室表”，点击确定完成创建。
1.3.2 按之前的方法修改查询，删除“序号”列，设置、保存并关闭科室表查询编辑窗口。
1.3.3 点击属性表的其他选项卡。将名称改为“科室表”。
1.3.4 在属性表的格式选项卡里，列数选择“3”，列宽设置为“0cm;0cm;2.542cm”，列标题选“是”。在属性表的数据选项卡的里，绑定列选:“3”。
1.4、 创建性别列表框，设置方法同地区表，选定字段性别。删除“序号”列，设置汇总，列数选择“1”，列宽设置为“2.542cm”，列标题选“是”。点击属性表的其他选项卡。将名称改为“性别表”。
1.5、 创建民族列表框，设置方法同上，将名称改为“民族表”。
1.6、 创建学历列表框，设置方法同上，将名称改为“学历表”。
1.6、 创建政治面貌列表框，设置方法同上，将名称改为“政治面貌表”。
1.7、 创建人员信息查询表列表框, 点击“≫”将可用字段(A)里除序号外的全部项目按顺序移动到选定字段(S)。
1.7.1 找到属性表的数据选项卡中的行来源，点击右边的“…”打开编辑界面，在右键菜单上选择。并保存关闭单位表查询窗口，在弹出的对话框中选择“是”保存SQL语句。
1.7.2 点击属性表的其他选项卡。将名称改为“人员信息查询表”。
1.7.3 在属性表的格式选项卡里，将列标题设置为“是”，调整列宽到合适的宽度，保证所有列都能完整显示，但总体不会超过列表框宽度。(可以用鼠标右键点击左上的人员信息查询选项卡，选择布局视图，可以立刻看到调整的结果。列数和绑定列不用调整。)
*  确保所有的列表框都要设置汇总，总计选项里选择“Group By”。所有列标题都要设置为“是”
二、设置查询
2.1、地区表的设置：鼠标选中地区表列表框，在属性表里的事件选项卡里，将单击的属性选择“[事件过程]”，点击“…”打开VBA编辑界面。
在Private Sub 地区表_Click()与End Sub的中间录入查询语句：注：1、“SECLCT XXXX XXXX ((()))；”语句的双引号和“；”必须要保留。2、SELECT查询语句不能直接转行，否则会报错，转行需要在末尾使用“ & _”不包括引号。
Dim strSQLa, strSQLb, strSQLc
strSQLa = "SELECT 人员信息表.区名, 人员信息表.单位 FROM 人员信息表 GROUP BY 人员信息表.区名, 人员信息表.单位 HAVING (((人员信息表.区名)=[forms]![人员信息查询]![地区表]));"  '根据地区表筛选单位表
With Me.单位表          '指定后续内容为当前窗口里的“单位表”列表框
.RowSource = strSQLa  'SECLCT赋值给“单位表”列表框
.Requery               '强制当前列表重新加载数据
End With                '结束对象指定
strSQLc = "SELECT 人员信息表.区名 FROM 人员信息表 GROUP BY 人员信息表.区名 HAVING (((人员信息表.区名)='0'));" '清空科室表内容
With Me.科室表
.RowSource = strSQLc
.Requery
End With
strSQLb = "SELECT 人员信息表.城市, 人员信息表.区名, 人员信息表.单位, 人员信息表.科室, 人员信息表.姓名, 人员信息表.年龄, 人员信息表.性别, 人员信息表.籍贯, 人员信息表.民族, 人员信息表.出生年月, 人员信息表.身份证号, 人员信息表.学历, 人员信息表.职务, 人员信息表.政治面貌 FROM 人员信息表 GROUP BY 人员信息表.序号, 人员信息表.城市, 人员信息表.区名, 人员信息表.单位, 人员信息表.科室, 人员信息表.姓名, 人员信息表.年龄, 人员信息表.性别, 人员信息表.籍贯, 人员信息表.民族, 人员信息表.出生年月, 人员信息表.身份证号, 人员信息表.学历, 人员信息表.职务, 人员信息表.政治面貌 HAVING (((人员信息表.区名)=[FORMS]![人员信息查询]![地区表]));"   '根据地区表，筛选人员信息
With Me.人员信息查询表
.RowSource = strSQLb
.Requery

* 注： 在VBA的编辑窗口中，SELECT 查询语句不能转行，并且必须写在“”内，如果“”内的内容出现双引号“”，需改为单引号‘’。
* 完成地区表列表框的设置后，可以在工具栏的视图里，将编辑视图改为窗体视图，选择不同的地区观测下方的查询表内容是否有变化。

将查询语句中的条件
HAVING (((人员信息表.区名)=[FORMS]![人员信息查询]![地区表]) and ((人员信息表.单位)=[FORMS]![人员信息查询]![单位表]) and ((人员信息表.科室)=[FORMS]![人员信息查询]![科室表]))

*  确保所有的列表框都要设置汇总，总计选项里选择“Group By”。所有列标题都要设置为“是”
1.8、在每一个列表框的下方插入一个文本框，文本框在设计选项卡中找到控件。用鼠标选中后，在窗体的空白区域拖动画出和列表框同样宽度的矩形，在弹出的窗口中不要进行任何设置，直接点击完成即可。
1.9、将文本框名称改为与上方的列表框对应的名称。例如：地区表下方的文本框改名为“地区”，单位表下方的文本框改名为“单位”，以此类推。* 注，所有名称均不包括双引号。
插入查询表及查询表的设置
2.1、用鼠标将左侧类别中表里的人员信息表拖到正在编辑的人员信息查询窗体中，在弹出的子窗体向导对话框中，将名字修改为“人员信息查询表”。在右侧的属性表里，将其他选项卡的名称修改为“人员信息查询表”，不包括双引号。
2.2、先用鼠标点击窗体空白处，再单击人员信息查询表。在右侧的属性表中，将水平定位点和垂直定位点均设置为两者，调整表格的右边和下边，距离窗体边缘约一格的距离。
2.3、点击人员信息查询表右上角的小方格，使小方格里出现一个黑色的小方块。在属性表的数据选项卡中，点击记录源右侧的“…”,在弹出的窗口中选择“是”，激活查询生成器，自动完成根据表建立的查询。
2.4、在弹窗的查询编辑窗口中，将人员信息表中，选中除“*”外的所有项目，拖放到下方的设置表中，完成后点击汇总。
2.5、在区名的条件里录入条件（Like FORMS![人员信息查询]![地区] & “*”）不包括括号，在单位的条件里录入条件（Like FORMS![人员信息查询]![单位] & “*”）不包括括号。其余的科室表、性别表、民族表、学历表和政治面貌表均按此方法录入对应的条件。完成后保存及关闭编辑界面。
设置查询条件和清除按钮
3.1、在设计选项卡中找到按钮控件，选择后在窗体空出用鼠标拉出矩形，选择取消完成按钮的创建（在这个设置里，不能选完成）。
3.2、修改按钮中的文字“Command45”（后面的数字会不一样），修改为“清除条件”，不包括双引号，属性表里的其他选项卡中，名称改为“清除条件”不包括双引号。
3.3、在右侧属性表的事件选项卡里，找到单击，通过下拉菜单修改为[事件过程]，并点击右侧的“…”打开VBA编辑界面。在清除条件的编辑区内，录入下列语句
Me.地区 = Null
Me.单位 = Null
Me.科室 = Null
Me.性别 = Null
Me.民族 = Null
Me.学历 = Null
Me.政治面貌 = Null
在VBA编辑界面的最下方，录入下列语句：
Function Ress()
Me.人员信息查询表.Requery
End Function
3.4、设置地区表列表框
鼠标选中地区表列表框，在右侧属性表的事件选项卡里，找到单击，通过下拉菜单修改为[事件过程]，并点击右侧的“…”打 开VBA编辑界面。在VBA的编辑界面中，在Private Sub 地区表_Click()和End Sub的中间空处，录入下列语句。
Dim strSQLa, strSQLb
strSQLa = "SELECT 人员信息表.区名, 人员信息表.单位 FROM 人员信息表 GROUP BY 人员信息表.区名, 人员信息表.单位 HAVING (((人员信息表.区名)=[forms]![人员信息查询]![地区表]));"  '根据地区表筛选单位表
With Me.单位表
.RowSource = strSQLa
.Requery
End With
strSQLb = "SELECT 人员信息表.区名 FROM 人员信息表 GROUP BY 人员信息表.区名 HAVING (((人员信息表.区名)='0'));" '清空科室表内容
With Me.科室表
.RowSource = strSQLb
.Requery
End With
Me.地区 = Me.地区表   '将自己的值赋给地区文本框
Ress '调用自定义函数
3.5、设置单位表列表框
鼠标选中单位表列表框，在右侧属性表的事件选项卡里，找到单击，通过下拉菜单修改为[事件过程]，并点击右侧的“…”打 开VBA编辑界面。在VBA的编辑界面中，在Private Sub 单位表_Click()和End Sub的中间空处，录入下列语句。
Dim strSQLa
strSQLa = "SELECT 人员信息表.区名,人员信息表.单位,人员信息表.科室 FROM 人员信息表 GROUP BY 人员信息表.区名,人员信息表.单位,人员信息表.科室 HAVING (((人员信息表.区名)=[forms]![人员信息查询]![地区表]) and  ((人员信息表.单位)=[forms]![人员信息查询]![单位表]));"
With Me.科室表
.RowSource = strSQLa
.Requery
End With
Me.单位 = Me.单位表
Ress  '调用自定义函数
3.7、其余的列表框设置
鼠标选中科室表列表框，在右侧属性表的事件选项卡里，找到单击，通过下拉菜单修改为[事件过程]，并点击右侧的“…”打 开VBA编辑界面。在VBA的编辑界面中，在Private Sub 科室表_Click()和End Sub的中间空处，输入下列语句。
Me.科室 = Me.科室表
Ress  '调用自定义函数
其余列表框的VBA设置方法与科室表方法一样，修改成对应的名称即可。
