#### 第一章 租用云端服务器 (VPS)

选一个数据中心，花几分钟完成注册和开机，这就是你 VPN 的「主机」。

**选择服务商**

- **支付宝友好：Vultr**
    
    界面最简单，支持支付宝。但注意：东京/热门地区 IP 容易被 GFW「连坐」封锁。
    
- **性价比：BandwagonHost (搬瓦工)**
    
    华人圈口碑老牌，支持支付宝。CN2 GIA 线路速度快，但价格略高（$50+/年）。中国用户扎堆，需注意选冷门机房。
    
- **永久免费：Oracle Cloud (甲骨文)**
    
    Always Free 方案送两台 ARM 小机，永久免费。但申请要求高（需真实信用卡），新号容易被收回。
    
- **新兴小厂：RackNerd / Hostinger**
    
    月付 $2-3 的超低价 VPS，适合预算有限。稳定性和 IP 品质不如大厂，但胜在便宜且易搭起来就用。
    

⚠️ **地区选择与 IP 被墙警告**

中国大陆用户推荐选择 **美国西海岸（洛杉矶/西雅图）、新加坡 或 日本大阪**（比东京干净）。尽量避开 Vultr 的日本东京节点——那里是重灾区，开出来的新 IP 很多直接就是被封锁的。如果第一次开出来 ping 不通，在 VPS 后台点「重建」换一个 IP 即可（通常不额外收费）。

🛒 **没有境外信用卡？**

以上所有服务商都可以在某宝搜索「 厂商名 + 充值 / 代开 / 礼品卡 」找到代付服务：

- **Vultr**：搜「Vultr 充值 / Vultr 礼品卡」，卖家直接帮你充进账户
    
- **DigitalOcean / Linode**：搜「DO 充值」「Linode 代开」，也可买「已开好 + 送 $100 试用金的成品号」
    
- **BandwagonHost**：原生支持支付宝，直接官网付款即可，不用绕道某宝
    

**1. 创建服务器（Deploy）**

在你的服务商后台创建一台新的服务器（Droplet/Instance），请务必按照以下配置选择：

|**选项**|**选什么**|**说明**|
|---|---|---|
|**System Image (系统)**|Ubuntu 22.04 LTS 或 24.04|最稳定的 Linux 版本|
|**Server Size (配置)**|$5~$6/mo（1 CPU, 1GB RAM）|VPN 用途完全够用|
|**Authentication (验证)**|Password (密码)|设置一个包含大小写和数字的复杂密码|

**2. 记录 IP 地址和密码**

开机完成后（通常需要 1 分钟），在控制台找到并记录以下两个信息：

- **IP Address**：形如 142.250.xx.xx 的一串数字
    
- **Password**：你刚才设置的密码（或者系统分配的 root 密码）

#### 第二章 远程连接到服务器

用 SSH 工具「进入」你的服务器，就像远程控制另一台电脑。

**根据你的系统选择工具**

**A. Windows 10 / 11 用户**

直接按 `Win + R`，输入 `cmd` 打开命令提示符，输入以下命令（把 IP 换成你的服务器 IP）：

`ssh root@你的服务器IP`

如果提示 `Are you sure you want to continue connecting?`，输入 `yes` 回车。

提示 `password:` 时粘贴你的密码（注意：输入时光标不会动，这是正常的），回车。

**B. macOS 用户 — 使用终端**

按 `⌘ Command + 空格` 搜索「终端」（Terminal）并打开，输入与 Windows 一样的命令并输入密码登录。

✅ 成功连接后，你会看到一个欢迎界面，并出现 `root@xxx:~#` 这样的提示符。这意味着你已经进入了服务器。

#### 第三章 安装 X-ui 面板 (VLESS+Reality)

只需执行一条一键安装脚本，剩下的所有配置都在可视化网页中用鼠标完成。

**1. 运行一键安装脚本**

确保你已看到 `root@...#` 提示符，复制以下命令，粘贴到终端窗口后按回车（这是目前 Github 最受欢迎的 3x-ui 分支）：

`bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)`

**2. 设置后台帐号密码**

安装过程中，屏幕会弹出提示让你进行基础设置（按提示输入）：

- `是否设置面板账户密码端口? [y/n]:` 输入 `y` 回车
    
- `请设置您的账户名:` 输入一个你记得住的名字，比如 `admin`
    
- `请设置您的密码:` 输入你的后台登录密码
    
- `请设置面板访问端口:` 输入一个 10000 到 65000 之间的数字，比如 `23456`
    
    完成后，系统会提示绿色的「x-ui 面板启动成功」。
    

**3. 登录网页控制面板**

打开你的浏览器，在网址栏输入 `http://你的服务器IP:你刚才设置的端口`（例如 `http://142.250.xx.xx:23456` ）。输入刚才设置的账号密码登录。

💡 **提示**

如果网页打不开，通常是因为服务商的云防火墙（Firewall / Security Group）未开放该端口。请去 VPS 后台的防火墙设置中放行该端口，或放行所有 TCP 端口。

**4. 创建 Reality 节点 (核心步骤)**

进入面板后，点击左侧菜单的 **面板设置 (Inbounds)**，点击 **+ 添加入站 (Add Inbound)**，按照以下严格填写：

|**字段**|**填写内容**|
|---|---|
|**备注 (Remark)**|随便填，例如 MyReality|
|**协议 (Protocol)**|选择 vless|
|**端口 (Port)**|填 443（最真实的 HTTPS 端口）|
|**传输配置 (Reality)**|将 Reality 开关打开|
|**目标网站 (Dest)**|填 `www.microsoft.com:443` (这是我们要伪装的对象)|
|**Server Names (SNI)**|填 `www.microsoft.com`|
|**密钥 (Keys)**|点击旁边的 Get New Cert 自动生成一对密钥|

检查无误后，点击右下角的 **添加 (Add)**。

**5. 获取一键导入链接**

在 Inbounds 列表中，找到你刚才创建的节点，点击左侧的 **二维码图标**（或操作里的分享按钮），点击 **复制 (Copy)**。你会得到一串以 `vless://` 开头的长代码。这就是你的终极「科学上网钥匙」。

#### 第四章 在你的设备上配置客户端

由于 Reality 是最新协议，老旧的客户端无法识别。请务必使用以下指定的现代客户端。

**各平台推荐客户端配置**

**1. Windows — v2rayN / Clash Verge Rev**

_推荐 v2rayN：_

- 前往 GitHub 下载 `v2rayN-Core.zip` 并解压打开。
    
- 确保剪贴板里已经复制了你的 `vless://` 链接。
    
- 在软件界面按 `Ctrl + V` 直接导入。
    
- 选中节点，按回车（测试延迟），然后在右下角任务栏图标右键 -> 系统代理 -> 自动配置系统代理 即可上网。
    

_Clash Verge Rev (更美观，支持分流)：_ 前往其 GitHub 下载，左侧点击「代理」，右上角「新建」->「从剪贴板导入」。

**2. Android — v2rayNG**

- 在 Google Play 或 Github 下载 v2rayNG。
    
- 打开 App，点击右上角的 `+` 号。
    
- 选择 **从剪贴板导入**（或扫描网页后台的二维码）。
    
- 点击右下角的 V 字圆圈图标启动连接。
    

**3. iOS — Shadowrocket (小火箭)**

⚠️ 需要美区 Apple ID

你需要登录美区的 App Store，购买并下载 Shadowrocket (约 $2.99)。

- 复制 `vless://` 链接后，打开 Shadowrocket。
    
- 软件会自动识别剪贴板内容，弹出提示，点击 **添加**。
    
- 回到首页，打开顶部的连接开关，完成 VPN 配置授权即可。

#### 第五章 验证连接 & 常见问题排错

Reality 协议虽然抗封锁极强，但配置稍有不慎就会连不上，请对照排查。

**1. 验证 VPN 是否生效**

打开浏览器访问以下网址，查看显示的 IP 地址：

`https://whatismyipaddress.com`

如果显示的 IP 与你服务器的 IP 一致，说明 Reality 隧道已建立。✅

**2. 最常见的 3 个翻车点**

|**症状**|**可能原因**|**解决方法**|
|---|---|---|
|**客户端报错：certificate has expired**|系统时间不对|Reality 对时间校验极严格。请确保你的电脑/手机时间与北京标准时间误差不超过 1 分钟。|
|**能上网，但几天后突然断联**|SNI 伪装被针对|登录 X-ui 后台，把 SNI 和 Dest 改为与你服务器同一云厂商的网址（如甲骨文云用 [www.oracle.com](https://www.oracle.com/)）以应对 IP 库查杀。|
|**X-ui 后台网页打不开**|云服务商防火墙拦截|登录 VPS 提供商控制台，在 Firewall 或 Security Group 中添加入站规则，放行你在步骤三设置的端口。|

🎉 **大功告成！**

你现在拥有了一个具备 2026 年最高抗封锁等级的私有网络节点。

可以随时登录 X-ui 网页后台，查看流量使用情况，或为家人朋友分配独立的连接链接。

只要你不使用它进行大流量 BT 下载或滥发垃圾邮件，这个节点可以长期稳定存活。

### 成本评估分析

根据教程中提供的信息，搭建此服务的成本主要分为**固定服务器成本**、**一次性/可选开销**以及**时间成本**。

#### 1. 服务器成本（核心开销）

根据你选择的 VPS 服务商不同，成本跨度很大：

- **极致零成本方案 (Oracle Cloud)：** `免费`。但获取门槛极高，需要真实的境外信用卡进行验证，且存在封号回收风险。
    
- **极致低价方案 (RackNerd / Hostinger 等)：** 约 `$2 ~ $3 / 月`（折合人民币约 ¥15 - ¥22 / 月）。适合预算极度有限的人群，但网络质量和稳定性相对一般。
    
- **主流稳定方案 (Vultr / DigitalOcean / Linode)：** 约 `$5 ~ $6 / 月`（折合人民币约 ¥35 - ¥45 / 月）。教程推荐的配置（1 CPU, 1GB RAM），网络质量尚可，操作对新手友好。
    
- **高端优质方案 (BandwagonHost/搬瓦工)：** 约 `$50+ / 年`（折合人民币约 ¥360+ / 年，即 ¥30+ / 月）。CN2 GIA 线路，晚高峰速度最快、延迟最低。
    

#### 2. 一次性 / 可选开销

- **代充服务手续费（如无境外信用卡）：** 购买约 $10 额度的礼品卡需花费约 `¥70`。按照实际汇率折算，代充卖家通常会收取 ¥10-¥15 左右的溢价手续费。
    
- **邮箱获取成本（如无可用邮箱）：** 购买 Gmail 等成品海外邮箱，约 `¥5 - ¥15` 一次性买断。
    
- **iOS 客户端软件购买：** 如果使用 iPhone/iPad，需要购买 Shadowrocket，价格为 `$2.99`（约 `¥22`，一次性买断）。Android 和 Windows/Mac 客户端均为免费开源。
    

#### 3. 时间与精力成本

- **配置时间：** 教程评估约为 `30 分钟`。
    

**综合成本总结：**

- **如果你是 Android/Windows 用户且自己有卡有邮箱：** 每月仅需支付 VPS 费用（通常在 `¥15 ~ ¥40 / 月`）。
    
- **如果你是纯“三无”小白（无卡、无邮箱）且使用 iPhone：** 启动资金大约需要 `¥100 左右`（含 ¥70 的 VPS 首充额度 + ¥10 邮箱 + ¥22 小火箭软件），之后每月维持固定服务器开销即可。
