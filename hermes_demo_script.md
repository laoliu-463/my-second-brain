# Hermes Agent 演示脚本

## Demo 0：开场（30秒）

**说："今天我来点不一样的——让我先问问 Hermes，我是谁。"**

→ 打开 QQ/任意平台，发消息：

```
你好 Herme，说说我是谁
```

→ 等它回复后："它居然知道我的名字/信息"

**目的：** 第一秒就抓住注意力——"这玩意儿怎么知道？""

---

## Demo 1：多平台无缝切换（2分钟）

**说："大家看，我同时在 QQ、飞书、Telegram 都登着。现在我在 QQ 上让它帮我找个资料。"**

1. 在 QQ 发消息：
```
帮我搜一下最近 AI Agent 领域有什么大事
```

2. 等待 Hermes 回复一段文字

3. **切换到飞书**，问：
```
刚才我让你搜的，总结成 3 点
```

**说："看，我换了个平台，但它记得我们聊过什么。这就是跨会话记忆。"**

---

## Demo 2：现场写代码（2分钟）

**说："来点硬核的——让它帮我写一段代码，然后直接跑起来。"**

1. 发消息：
```
用 Python 写一个快速排序算法，然后运行测试
```

2. 展示结果（代码 + 运行输出）

3. **追加问题**：
```
改成链表版本的快速排序
```

**说："它不只是写代码，还能执行。这就是 AI Coding Agent 和 ChatGPT 的本质区别。"**

---

## Demo 3：文件处理 + 演讲稿生成（1分钟）

**说："最后一个小功能——我让它帮我整理一下刚才聊的内容，做成要点清单。"**

1. 发消息：
```
把我们今天的对话整理成 5 个要点，用来演讲用
```

2. 展示结果

**说："看到了吗？它能做的不只是聊天，而是真正帮你完成任务。"**

---

## 收尾（30秒）

**说："从 2023 年的 CLI 工具，到现在的多平台 Agent，Hermes 走过了 4 年。但最让我震撼的不是技术本身——而是 AI 真的开始'动手做事'了，不是只会说话。"**

**"大家有什么问题吗？"**
